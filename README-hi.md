## कहानी — EurekAI क्यों?

**EurekAI** [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([आधिकारिक साइट](https://worldwide-hackathon.mistral.ai/)) (मार्च 2026) के दौरान जन्मा। मुझे एक विषय चाहिए था — और विचार कुछ बहुत ठोस चीज़ से आया: मैं नियमित रूप से अपनी बेटी के साथ परीक्षाओं की तैयारी करता हूँ, और मैंने सोचा कि AI की मदद से इसे और अधिक मज़ेदार और इंटरैक्टिव बनाना संभव होना चाहिए।

उद्देश्य: **किसी भी इनपुट** को लें — पाठ का फोटो, कॉपी-पेस्ट किया हुआ टेक्स्ट, वॉइस रिकॉर्डिंग, वेब खोज — और उसे **संशोधन शीट, फ्लैशकार्ड, क्विज़, पॉडकास्ट, रिक्त-स्थान वाले पाठ, चित्रण, और भी बहुत कुछ** में बदल दें। यह सब Mistral AI के फ्रेंच मॉडलों द्वारा संचालित, जिससे यह फ्रेंच-भाषी छात्रों के लिए स्वाभाविक रूप से उपयुक्त समाधान बनता है।

[प्रारंभिक प्रोटोटाइप](https://github.com/jls42/worldwide-hackathon.mistral.ai) हैकाथॉन के दौरान Mistral सेवाओं पर केंद्रित एक प्रूफ-ऑफ-कॉन्सेप्ट के रूप में 48 घंटों में बनाया गया था — पहले से ही कार्यात्मक, लेकिन सीमित। तब से, EurekAI एक वास्तविक परियोजना बन गई है: रिक्त-स्थान वाले पाठ, अभ्यासों में नेविगेशन, वेब स्क्रैपिंग, कॉन्फ़िगर करने योग्य पैरेंटल मॉडरेशन, गहन कोड समीक्षा, और बहुत कुछ। पूरा कोड AI द्वारा उत्पन्न है — मुख्य रूप से [Claude Code](https://code.claude.com/), कुछ योगदानों के साथ [Codex](https://openai.com/codex/) और [Gemini CLI](https://geminicli.com/) के माध्यम से।

## गहराई से देखें — विशेषताएँ

### बहु-मोडल इनपुट

EurekAI प्रोफ़ाइल के अनुसार मॉडरेट किए गए 4 प्रकार के स्रोत स्वीकार करता है (बच्चे और किशोर के लिए डिफ़ॉल्ट रूप से सक्रिय) :

- **फ़ाइल आयात** — JPG, PNG या PDF फ़ाइलें `mistral-ocr-latest` द्वारा संसाधित (मुद्रित पाठ, तालिकाएँ, हस्तलिखित लेखन), या टेक्स्ट फ़ाइलें (TXT, MD) सीधे आयात की जाती हैं।
- **मुक्त पाठ** — कोई भी सामग्री टाइप या पेस्ट करें। यदि मॉडरेशन सक्रिय है तो संग्रहण से पहले मॉडरेट किया जाता है।
- **ध्वनि इनपुट** — ब्राउज़र में ऑडियो रिकॉर्ड करें। `voxtral-mini-latest` द्वारा ट्रांसक्राइब किया गया। `language="fr"` पैरामीटर पहचान को अनुकूलित करता है।
- **वेब / URL** — सामग्री को सीधे स्क्रैप करने के लिए एक या अधिक URLs पेस्ट करें (JS पेजों के लिए Readability + Lightpanda), या Mistral Agent के माध्यम से वेब खोज के लिए कीवर्ड टाइप करें। एकल फ़ील्ड दोनों स्वीकार करता है — URLs और कीवर्ड स्वचालित रूप से अलग किए जाते हैं, प्रत्येक परिणाम एक स्वतंत्र स्रोत बनाता है।

| जनरेटर | मॉडल | आउटपुट |
|---|---|---|
| **संशोधन शीट** | `mistral-large-latest` | शीर्षक, सारांश, मुख्य बिंदु, शब्दावली, उद्धरण, किस्सा |
| **फ्लैशकार्ड** | `mistral-large-latest` | स्रोत संदर्भों के साथ Q/A कार्ड (कॉन्फ़िगर करने योग्य संख्या) |
| **MCQ क्विज़** | `mistral-large-latest` | बहुविकल्पीय प्रश्न, स्पष्टीकरण, अनुकूलनीय समीक्षा (कॉन्फ़िगर करने योग्य संख्या) |
| **रिक्त-स्थान वाले पाठ** | `mistral-large-latest` | संकेतों के साथ पूरा करने योग्य वाक्य, सहिष्णु सत्यापन (Levenshtein) |

### चैट-आधारित AI ट्यूटर

पाठ्य दस्तावेज़ों तक पूर्ण पहुँच वाला एक संवादात्मक ट्यूटर:

- `mistral-large-latest` का उपयोग करता है
- **टूल-कॉलिंग** : बातचीत के दौरान शीट, फ्लैशकार्ड, क्विज़ या रिक्त-स्थान वाले पाठ बना सकता है
- प्रति पाठ 50 संदेशों का इतिहास
- प्रोफ़ाइल के लिए सक्रिय होने पर सामग्री मॉडरेशन

### स्वचालित राउटर

राउटर स्रोत सामग्री का विश्लेषण करने और 7 उपलब्ध जनरेटरों में से सबसे प्रासंगिक जनरेटरों का सुझाव देने के लिए `mistral-small-latest` का उपयोग करता है। इंटरफ़ेस वास्तविक समय में प्रगति दिखाता है: पहले विश्लेषण चरण, फिर व्यक्तिगत जनरेशन जिनमें रद्द करना संभव है।

### अनुकूली सीखना

- **क्विज़ आँकड़े** : प्रत्येक प्रश्न के लिए प्रयासों और सटीकता का ट्रैक रखना
- **क्विज़ समीक्षा** : कमजोर अवधारणाओं को लक्षित करने वाले 5-10 नए प्रश्न उत्पन्न करता है
- **निर्देश पहचान** : समीक्षा निर्देश ("मैं अपना पाठ तब जानता हूँ जब मैं...") का पता लगाता है और उन्हें संगत पाठ्य जनरेटरों (शीट, फ्लैशकार्ड, क्विज़, रिक्त-स्थान वाले पाठ) में प्राथमिकता देता है

### सुरक्षा और माता-पिता नियंत्रण

- **4 आयु समूह** : बच्चा (≤10 वर्ष), किशोर (11-15), छात्र (16-25), वयस्क (26+)
- **सामग्री मॉडरेशन** : `mistral-moderation-latest` के साथ 10 उपलब्ध श्रेणियाँ, बच्चे/किशोर के लिए डिफ़ॉल्ट रूप से 5 अवरुद्ध (`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`)। सेटिंग्स में प्रोफ़ाइल के अनुसार श्रेणियाँ अनुकूलन योग्य।
- **पैरेंटल PIN** : SHA-256 हैश, 15 वर्ष से कम उम्र के प्रोफ़ाइल के लिए आवश्यक। उत्पादन परिनियोजन के लिए, साल्ट के साथ धीमा हैश (Argon2id, bcrypt) उपयोग करें।
- **चैट प्रतिबंध** : 16 वर्ष से कम के लिए AI चैट डिफ़ॉल्ट रूप से निष्क्रिय, माता-पिता द्वारा सक्रिय किया जा सकता है

### बहु-प्रोफ़ाइल प्रणाली

- नाम, आयु, अवतार, भाषा प्राथमिकताओं के साथ कई प्रोफ़ाइल
- `profileId` के माध्यम से प्रोफ़ाइलों से जुड़े प्रोजेक्ट
- कैस्केडिंग हटाना: एक प्रोफ़ाइल हटाने पर उसके सभी प्रोजेक्ट हट जाते हैं

### बहु-प्रदाता TTS और कस्टम आवाज़ें

- **Mistral Voxtral TTS** (डिफ़ॉल्ट) : `voxtral-mini-tts-latest`, किसी अतिरिक्त कुंजी की आवश्यकता नहीं
- **ElevenLabs** (वैकल्पिक) : `eleven_v3`, प्राकृतिक आवाज़ें, `ELEVENLABS_API_KEY` की आवश्यकता
- ऐप की सेटिंग्स में कॉन्फ़िगर करने योग्य प्रदाता
- **कस्टम आवाज़ें** : माता-पिता Mistral Voices API के माध्यम से (ऑडियो नमूने से) अपनी खुद की आवाज़ें बना सकते हैं और उन्हें होस्ट/अतिथि भूमिकाओं को असाइन कर सकते हैं — तब पॉडकास्ट और ध्वनि क्विज़ माता-पिता की आवाज़ में पढ़े जाते हैं, जिससे बच्चे के लिए अनुभव और भी अधिक immersive हो जाता है
- दो कॉन्फ़िगर करने योग्य आवाज़ भूमिकाएँ: **होस्ट** (मुख्य कथावाचक) और **अतिथि** (पॉडकास्ट की दूसरी आवाज़)
- Mistral आवाज़ों की पूर्ण सूची सेटिंग्स में उपलब्ध, भाषा के अनुसार फ़िल्टर करने योग्य

### अंतर्राष्ट्रीयकरण

- इंटरफ़ेस 9 भाषाओं में उपलब्ध: fr, en, es, pt, it, nl, de, hi, ar

## त्वरित शुरुआत

```bash
# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install

# Configurer les clés API
cp .env.example .env
# Éditez .env avec vos clés :
#   MISTRAL_API_KEY=<your_api_key>           (requis)
#   ELEVENLABS_API_KEY=<your_api_key>        (optionnel, TTS alternatif)
#   SONAR_TOKEN=...                          (optionnel, CI SonarCloud uniquement)

# Lancer le développement
npm run dev
# → Backend :  http://localhost:3000 (API)
# → Frontend : http://localhost:5173 (serveur Vite avec HMR)
```

> **नोट** : Mistral Voxtral TTS डिफ़ॉल्ट प्रदाता है — `MISTRAL_API_KEY` के अलावा किसी अतिरिक्त कुंजी की आवश्यकता नहीं। ElevenLabs एक वैकल्पिक TTS प्रदाता है जिसे सेटिंग्स में कॉन्फ़िगर किया जा सकता है।

## कंटेनर के साथ परिनियोजन

छवि **GitHub Container Registry** पर प्रकाशित है:

```bash
# Télécharger l'image
podman pull ghcr.io/jls42/eurekai:latest

# Lancer EurekAI
mkdir -p ./data
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=<your_api_key> \
  -e ELEVENLABS_API_KEY=<your_api_key> \
  -v ./data:/app/output:U \
  -p 3000:3000 \
  ghcr.io/jls42/eurekai:latest
# → http://localhost:3000
```

> **`:U`** एक rootless Podman फ़्लैग है जो स्वचालित रूप से वॉल्यूम अनुमतियाँ समायोजित करता है।
> **`ELEVENLABS_API_KEY`** वैकल्पिक है (वैकल्पिक TTS)।

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

| **मल्टी-जनरेशन सिस्टम** | हर जनरेशन अपने स्वयं के ID के साथ एक स्वतंत्र ऑब्जेक्ट है — एक ही कोर्स के लिए कई शीट, क्विज़ आदि की अनुमति देता है। |
| **आयु-अनुकूल प्रॉम्प्ट्स** | 4 आयु समूह, अलग-अलग शब्दावली, जटिलता और टोन के साथ — वही सामग्री सीखने वाले के अनुसार अलग तरह से सिखाती है। |
| **Agents-आधारित सुविधाएँ** | छवि जनरेशन और वेब खोज अस्थायी Mistral Agents का उपयोग करते हैं — स्वचालित सफ़ाई के साथ स्वच्छ जीवनचक्र। |
| **स्मार्ट URL स्क्रैपिंग** | एक ही फ़ील्ड में URLs और मिश्रित कीवर्ड स्वीकार किए जाते हैं — URLs को Readability (स्थिर पृष्ठ) के माध्यम से स्क्रैप किया जाता है, Lightpanda fallback (JS/SPA पृष्ठ) के साथ; कीवर्ड Mistral web_search Agent को ट्रिगर करते हैं। प्रत्येक परिणाम एक स्वतंत्र स्रोत बनाता है। |
| **मल्टी-प्रदाता TTS** | डिफ़ॉल्ट रूप से Mistral Voxtral TTS (अतिरिक्त कुंजी नहीं), वैकल्पिक रूप से ElevenLabs — पुनःस्टार्ट के बिना कॉन्फ़िगर करने योग्य। |
