## कहानी — EurekAI क्यों?

**EurekAI** का जन्म [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([आधिकारिक साइट](https://worldwide-hackathon.mistral.ai/)) (मार्च 2026) के दौरान हुआ था। मुझे एक आइडिया चाहिए था — और विचार कुछ बहुत ही ठोस से आया: मैं नियमित रूप से अपनी बेटी के साथ टेस्ट की तैयारी करता/करती हूं, और मैंने सोचा कि IA की मदद से इसे और अधिक मजेदार और इंटरैक्टिव बनाया जा सकता है।

लक्ष्य: किसी भी इनपुट — एक पाठ की तस्वीर, कॉपी-पेस्ट किया गया टेक्स्ट, एक वॉइस रिकॉर्डिंग, वेब सर्च — को लें और उसे **रिवीजन नोट्स, फ्लैशकार्ड, क्विज़, पॉडकास्ट, क्लोज़-टेक्स्ट (रिक्त स्थान वाले टेक्स्ट), इलस्ट्रेशन, और भी बहुत कुछ** में बदल दें। यह सब Mistral AI के फ्रेंच मॉडल से संचालित है, इसलिए यह फ्रेंच बोलने वाले छात्रों के लिए स्वाभाविक रूप से अनुकूल समाधान बनता है।

प्रारंभिक [प्रोटोटाइप](https://github.com/jls42/worldwide-hackathon.mistral.ai) 48 घंटे के हैकाथॉन में Mistral सेवाओं के आस-पास proof-of-concept के रूप में बनाया गया था — पहले से कार्यशील, पर सीमित। तब से, EurekAI एक वास्तविक प्रोजेक्ट बन चुका है: क्लोज़-टेक्स्ट, एक्सरसाइज में नेविगेशन, वेब स्क्रैपिंग, कॉन्फ़िगरेबल पैरेंटल मॉडरेशन, गहन कोड रिव्यू, और भी बहुत कुछ। संपूर्ण कोड IA द्वारा जेनरेट किया गया है — मुख्य रूप से [Claude Code](https://code.claude.com/), कुछ योगदान [Codex](https://openai.com/codex/) और [Gemini CLI](https://geminicli.com/) के जरिए।

## विस्तृत विवरण — सुविधाएँ

### मल्टी-मोडल इनपुट

EurekAI 4 प्रकार के स्रोत स्वीकार करता है, प्रोफ़ाइल के अनुरूप मॉडरेट किए जाते हैं (डिफ़ॉल्ट रूप से बच्चा और किशोर पर सक्रिय):

- **फाइल आयात** — JPG, PNG या PDF फाइलें `mistral-ocr-latest` द्वारा प्रोसेस की जाती हैं (प्रिंटेड टेक्स्ट, टेबल, हैंडराइटिंग), या टेक्स्ट फाइलें (TXT, MD) सीधे इम्पोर्ट होती हैं।
- **फ्री टेक्स्ट** — किसी भी सामग्री को टाइप या पेस्ट करें। यदि मॉडरेशन सक्रिय है तो स्टोरेज से पहले मॉडरेट किया जाता है।
- **वॉइस इनपुट** — ब्राउज़र में ऑडियो रिकॉर्ड करें। `voxtral-mini-latest` द्वारा ट्रांसक्राइब किया जाता है। `language="fr"` पैरामीटर रिकग्निशन को ऑप्टिमाइज़ करता है।
- **वेब / URL** — एक या कई URLs पेस्ट करें ताकि कंटेंट सीधे स्क्रैप हो (Readability + Lightpanda JS पेजों के लिए), या कीवर्ड टाइप करके Agent Mistral के माध्यम से वेब सर्च करें। यह यूनिक फील्ड दोनों स्वीकार करता है — URLs और कीवर्ड्स स्वतः अलग किये जाते हैं, हर रिज़ल्ट एक स्वतंत्र स्रोत बनाता है।

### IA कंटेंट जेनरेशन

सात प्रकार के सीखने के मटेरियल जेनरेट किए जाते हैं:

| जनरेटर | मॉडल | आउटपुट |
|---|---|---|
| **रिवीजन नोट** | `mistral-large-latest` | शीर्षक, सारांश, प्रमुख बिंदु, शब्दावली, उद्धरण, जानकारी |
| **फ्लैशकार्ड** | `mistral-large-latest` | स्रोत रेफरेंस के साथ Q/A कार्ड्स (गिनती समायोज्य) |
| **MCQ क्विज़** | `mistral-large-latest` | बहुविकल्पीय प्रश्न, स्पष्टीकरण, अनुकूलन योग्य समीक्षा (गिनती समायोज्य) |
| **क्लोज़-टेक्स्ट** | `mistral-large-latest` | संकेतों के साथ पूरा करने के वाक्य, टॉलरेंट वैलिडेशन (Levenshtein) |
| **पॉडकास्ट** | `mistral-large-latest` + Voxtral TTS | 2-आवाज़ स्क्रिप्ट → MP3 ऑडियो |
| **चित्रण** | Agent `mistral-large-latest` | टूल `image_generation` के माध्यम से शैक्षिक इमेज |
| **वॉइस क्विज़** | `mistral-large-latest` + Voxtral TTS + STT | प्रश्न TTS → उत्तर STT → IA सत्यापन |

### चैट के जरिए IA ट्यूटर

एक संवादात्मक ट्यूटर जो कोर्स डॉक्यूमेंट्स तक पूर्ण एक्सेस रखता है:

- उपयोग करता है `mistral-large-latest`
- **टूल कॉल**: बातचीत के दौरान फिचर, फ्लैशकार्ड, क्विज़ या क्लोज़-टेक्स्ट जेनरेट कर सकता है
- प्रति कोर्स 50 संदेशों का इतिहास
- प्रोफ़ाइल सक्रिय होने पर कंटेंट की मॉडरेशन

### ऑटोमैटिक राउटर

राउटर कंटेंट के स्रोतों का विश्लेषण करने के लिए `mistral-small-latest` का उपयोग करता है और 7 उपलब्ध जेनरेटरों में से सबसे उपयुक्त सुझाव देता है। इंटरफ़ेस रीयल-टाइम प्रगति दिखाता है: पहले एक विश्लेषण चरण, फिर व्यक्तिगत जेनरेशन्स जिनमें रद्द करने का विकल्प होता है।

### अनुकूली सीखाई

- **क्विज़ आँकड़े**: प्रश्नों के प्रयास और सटीकता का ट्रैक
- **क्विज़ रिवीजन**: कमजोर कॉन्सेप्ट्स पर लक्षित 5-10 नए प्रश्न जेनरेट करता है
- **निर्देश पहचान**: रिवीजन निर्देशों का पता लगाता है ("मैं अपनी पाठ याद रखता/रखती हूँ अगर मैं...") और इन्हें टेक्स्ट जेनरेटरों (नोट, फ्लैशकार्ड, क्विज़, क्लोज़-टेक्स्ट) में प्राथमिकता देता है

### सुरक्षा एवं पैरेंटल कंट्रोल

- **4 आयु समूह**: बच्चा (≤10 वर्ष), किशोर (11-15), छात्र (16-25), वयस्क (26+)
- **कंटेंट मॉडरेशन**: `mistral-moderation-latest` जिसमें 10 उपलब्ध कैटेगरी हैं, डिफ़ॉल्ट रूप से बच्चा/किशोर के लिए 5 ब्लॉक किए जाते हैं (`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`)। प्रोफ़ाइल सेटिंग्स में कैटेगरी कस्टमाइज़ेबल हैं।
- **पैरेंटल PIN**: SHA-256 हैश, 15 साल से कम प्रोफ़ाइल के लिए आवश्यक। प्रोडक्शन डिप्लॉयमेंट के लिए धीमे हैश और सॉल्ट (Argon2id, bcrypt) का उपयोग सुझाया जाता है।
- **चैट प्रतिबंध**: 16 साल से कम के लिए IA चैट डिफ़ॉल्ट रूप से अक्षम; माता-पिता द्वारा सक्रिय किया जा सकता है

### मल्टी-प्रोफ़ाइल सिस्टम

- कई प्रोफ़ाइल जिनमें नाम, आयु, अवतार, भाषा प्राथमिकताएँ
- प्रोफ़ाइल से जुड़े प्रोजेक्ट्स `profileId` के जरिए
- कास्केड डिलीशन: प्रोफ़ाइल हटाने पर उसके सभी प्रोजेक्ट्स भी हटते हैं

### मल्टी-प्रोवाइडर TTS और कस्टम वॉइस

- **Mistral Voxtral TTS** (डिफ़ॉल्ट): `voxtral-mini-tts-latest`, अतिरिक्त क्लिप की ज़रूरत नहीं
- **ElevenLabs** (वैकल्पिक): `eleven_v3`, नैचुरल वॉइसेस, ज़रूरत है `ELEVENLABS_API_KEY`
- प्रोवाइडर एप सेटिंग्स में कॉन्फ़िगरेबल
- **कस्टम वॉइस**: माता-पिता Mistral Voices API के जरिए अपने स्वयं के वॉइस सैंपल से बना सकते हैं और इन्हें होस्ट/गेस्ट रोल्स पर असाइन कर सकते हैं — पॉडकास्ट और वॉइस क्विज़ तब माता-पिता की आवाज़ से पढ़े जाते हैं, जिससे बच्चे के लिए अनुभव और भी अधिक इमर्सिव बनता है
- दो वॉइस रोल कॉन्फ़िगरेबल: **होस्ट** (मुख्य कथावाचक) और **गेस्ट** (पॉडकास्ट की दूसरी आवाज़)
- Mistral की पूरी वॉइस सूची सेटिंग्स में उपलब्ध, भाषा के अनुसार फ़िल्टर करने योग्य

### अंतरराष्ट्रीयकरण

- इंटरफ़ेस 9 भाषाओं में उपलब्ध: fr, en, es, pt, it, nl, de, hi, ar
- IA प्रम्प्ट 15 भाषाओं का समर्थन करते हैं (fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv)

## त्वरित शुरुआत

```bash
# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install

# Configurer les clés API
cp .env.example .env
# Éditez .env avec vos clés :
#   MISTRAL_API_KEY=votre_clé_ici           (requis)
#   ELEVENLABS_API_KEY=votre_clé_ici        (optionnel, TTS alternatif)
#   SONAR_TOKEN=...                          (optionnel, CI SonarCloud uniquement)

# Lancer le développement
npm run dev
# → Backend :  http://localhost:3000 (API)
# → Frontend : http://localhost:5173 (serveur Vite avec HMR)
```

> **नोट** : Mistral Voxtral TTS डिफ़ॉल्ट प्रोवाइडर है — `MISTRAL_API_KEY` के अतिरिक्त किसी और की आवश्यकता नहीं। ElevenLabs वैकल्पिक TTS प्रोवाइडर है जिसे सेटिंग्स में कॉन्फ़िगर किया जा सकता है।

## कंटेनर के साथ परिनियोजन

इमेज **GitHub Container Registry** पर प्रकाशित है:

```bash
# Télécharger l'image
podman pull ghcr.io/jls42/eurekai:latest

# Lancer EurekAI
mkdir -p ./data
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=votre_clé_ici \
  -e ELEVENLABS_API_KEY=votre_clé_ici \
  -v ./data:/app/output:U \
  -p 3000:3000 \
  ghcr.io/jls42/eurekai:latest
# → http://localhost:3000
```

> **`:U`** एक Podman rootless फ्लैग है जो वॉल्यूम के परमिशंस को स्वतः समायोजित करता है।
> **`ELEVENLABS_API_KEY`** वैकल्पिक है (वैकल्पिक TTS)।

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

| **उम्र के हिसाब से अनुकूल प्रॉम्प्ट्स** | 4 आयु समूह vocab, जटिलता और टोन के साथ भिन्न — एक ही कंटेंट अलग तरीके से सिखाता है। |
| **एजेंट-आधारित फ़ीचर** | इमेज जनरेशन और वेब सर्च अस्थायी Mistral एजेंट्स का उपयोग करते हैं — लाइफसाइकल क्लीनअप के साथ स्वच्छ। |
| **स्मार्ट URL स्क्रैपिंग** | एक ही फ़ील्ड URLs और कीवर्ड मिलाकर स्वीकार करता है — URLs को Readability (स्टैटिक पेज) द्वारा स्क्रैप किया जाता है और JS/SPA पेजों के लिए Lightpanda fallback, कीवर्ड्स Mistral web_search एजेंट ट्रिगर करते हैं। हर परिणाम एक स्वतंत्र स्रोत बनाता है। |
| **मल्टी-प्रोवाइडर TTS** | डिफ़ॉल्ट तौर पर Mistral Voxtral TTS (अतिरिक्त कुंजी नहीं), वैकल्पिक रूप से ElevenLabs — बिना रिस्टार्ट के कॉन्फ़िगर करने योग्य। |
