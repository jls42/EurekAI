## कहानी — EurekAI क्यों?

**EurekAI** का जन्म [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([आधिकारिक साइट](https://worldwide-hackathon.mistral.ai/)) के दौरान हुआ (मार्च 2026)। मुझे एक विषय चाहिए था — और यह विचार कुछ बहुत ही ठोस से आया: मैं नियमित रूप से अपनी बेटी के साथ टेस्ट की तैयारी करता हूँ, और मैंने सोचा कि IA (AI) की मदद से इसे और अधिक मजेदार और इंटरैक्टिव बनाया जा सकता है।

लक्ष्य: किसी भी इनपुट को लें — पाठ का फोटो, कॉपी-पेस्ट किया हुआ टेक्स्ट, वॉइस रिकॉर्डिंग, वेब खोज — और उसे बदलें **रिवीजन नोट्स, फ्लैशकार्ड, क्विज़, पॉडकास्ट, रिक्त स्थान वाला टेक्स्ट, चित्र, और भी बहुत कुछ** में। यह सब Mistral AI के फ्रेंच मॉडल्स द्वारा संचालित होता है, जो इसे फ्रांसीसी भाषी छात्रों के लिए स्वाभाविक रूप से उपयुक्त बनाता है।

आरंभिक [प्रोटोटाइप](https://github.com/jls42/worldwide-hackathon.mistral.ai) हैकाथॉन में 48 घंटे में Mistral सेवाओं के चारों ओर प्रमाण-स्वरूप के रूप में बनाया गया था — पहले से काम कर रहा था, पर सीमित। तब से, EurekAI एक वास्तविक प्रोजेक्ट बन गया है: रिक्त स्थान वाले टेक्स्ट, अभ्यासों में नेविगेशन, वेब स्क्रैपिंग, कस्टमाइजेबल पैरेंटल मॉडरेशन, गहन कोड समीक्षा, और भी बहुत कुछ। पूरे प्रोजेक्ट का कोड AI द्वारा जनरेट किया गया है — मुख्य रूप से [Claude Code](https://code.claude.com/), कुछ योगदान [Codex](https://openai.com/codex/) और [Gemini CLI](https://geminicli.com/) के माध्यम से।

## गहराई में — फ़ीचर्स

### मल्टी-मॉडल इनपुट

EurekAI 4 प्रकार के स्रोत स्वीकार करता है, जो प्रोफ़ाइल के अनुसार मॉडरेट होते हैं (बच्चे और किशोर के लिए डिफ़ॉल्ट रूप से सक्रिय):

- **फ़ाइल इम्पोर्ट** — JPG, PNG या PDF फ़ाइलें `mistral-ocr-latest` द्वारा प्रोसेस की जाती हैं (छपा हुआ टेक्स्ट, टेबल, हस्तलिखित लेखन), या टेक्स्ट फ़ाइलें (TXT, MD) सीधे इम्पोर्ट की जाती हैं।
- **मुक्त टेक्स्ट** — कोई भी कंटेंट टाइप या पेस्ट करें। यदि मॉडरेशन सक्रिय है तो स्टोरेज से पहले मॉडरेट किया जाता है।
- **वॉइस इनपुट** — ब्राउज़र में ऑडियो रिकॉर्ड करें। `voxtral-mini-latest` द्वारा ट्रांसक्राइब किया जाता है। `language="fr"` पैरामीटर मान्यता को अनुकूलित करता है।
- **वेब / URL** — एक या अधिक URLs पेस्ट करें ताकि पेज की सामग्री सीधे स्क्रैप की जा सके (Readability + Lightpanda JS पेजों के लिए), या Agent Mistral के माध्यम से कीवर्ड लिखकर वेब खोज करें। एकल फ़ील्ड दोनों स्वीकार करता है — URLs और कीवर्ड्स स्वतः अलग कर दिए जाते हैं, हर परिणाम एक स्वतंत्र स्रोत बन जाता है।

### IA सामग्री निर्माण

सात प्रकार के अध्ययन सामग्री जेनरेट होते हैं:

| जेनरेटर | मॉडल | आउटपुट |
|---|---|---|
| **रिवीजन नोट** | `mistral-large-latest` | शीर्षक, सार, मुख्य बिंदु, शब्दावली, उद्धरण, उपाख्यान |
| **फ्लैशकार्ड** | `mistral-large-latest` | स्रोतों के संदर्भ के साथ Q/A कार्ड (संख्या कॉन्फ़िगर करने योग्य) |
| **MCQ क्विज़** | `mistral-large-latest` | बहुविकल्पीय प्रश्न, व्याख्याएँ, अनुकूलनशील रीव्यू (संख्या कॉन्फ़िगर करने योग्य) |
| **रिक्त स्थान वाले पाठ** | `mistral-large-latest` | संकेतों के साथ भरने योग्य वाक्य, सहनशील मान्यकरण (Levenshtein) |
| **पॉडकास्ट** | `mistral-large-latest` + Voxtral TTS | 2-आवाज़ स्क्रिप्ट → MP3 ऑडियो |
| **चित्रण** | Agent `mistral-large-latest` | टूल `image_generation` के माध्यम से शैक्षिक छवि |
| **वॉइस क्विज़** | `mistral-large-latest` + Voxtral TTS + STT | TTS द्वारा प्रश्न → STT द्वारा उत्तर → IA सत्यापन |

### चैट के माध्यम से AI ट्यूटर

क्लास डॉक्स तक पूर्ण पहुँच वाला एक संवादात्मक ट्यूटर:

- उपयोग करता है `mistral-large-latest`
- **टूल कॉल्स**: बातचीत के दौरान रिवीजन नोट, फ्लैशकार्ड, क्विज़ या रिक्त स्थान वाले टेक्स्ट जेनरेट कर सकता है
- प्रति कोर्स 50 संदेशों का इतिहास
- यदि प्रोफ़ाइल के लिए सक्रिय हो तो कंटेंट मॉडरेशन लागू

### स्वचालित राउटर

राउटर सामग्री का विश्लेषण करने के लिए `mistral-small-latest` का उपयोग करता है और उपलब्ध 7 जेनरेटरों में से सबसे प्रासंगिक सुझाव देता है। इंटरफ़ेस वास्तविक समय में प्रगति दिखाता है: पहले एक विश्लेषण चरण, फिर व्यक्तिगत जेनरेशन्स — जिन्हें रद्द भी किया जा सकता है।

### अनुकूली सीखना

- **क्विज़ आँकड़े**: प्रयासों और प्रश्न-दर-प्रश्न सटीकता का ट्रैक
- **क्विज़ रिवीजन**: कमजोर अवधारणाओं को लक्षित करते हुए 5-10 नए प्रश्न जनरेट करता है
- **निर्देश पहचान**: पुनरावृत्ति निर्देशों (जैसे "अगर मैं ... जानता हूँ तो मैं अपनी पाठ्यपुस्तक जानता हूँ") का पता लगाता है और उन्हें टेक्स्ट-जेनरेटरों (नोट, फ्लैशकार्ड, क्विज़, रिक्त स्थान टेक्स्ट) में प्राथमिकता देता है

### सुरक्षा और अभिभावकीय नियंत्रण

- **4 आयु समूह**: बच्चा (≤10 वर्ष), किशोर (11-15), छात्र (16-25), वयस्क (26+)
- **कंटेंट मॉडरेशन**: `mistral-moderation-latest` के साथ 10 श्रेणियाँ उपलब्ध; बच्चे/किशोर के लिए डिफ़ॉल्ट रूप से 5 ब्लॉक की गई हैं (`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`). प्रोफ़ाइल सेटिंग्स में श्रेणियाँ कस्टमाइज़ की जा सकती हैं।
- **पैरेंटल PIN**: SHA-256 हैश, 15 वर्ष से कम के प्रोफ़ाइल के लिए आवश्यक। प्रोडक्शन डिप्लॉयमेंट के लिए सॉल्ट के साथ स्लो हैश (Argon2id, bcrypt) की योजना रखें।
- **चैट प्रतिबंध**: 16 वर्ष से कम के लिए डिफ़ॉल्ट रूप से AI चैट अक्षम; माता-पिता द्वारा सक्षम किया जा सकता है

### मल्टी-प्रोफ़ाइल सिस्टम

- कई प्रोफ़ाइल नाम, आयु, अवतार, भाषा प्राथमिकताएँ के साथ
- प्रोफ़ाइल से जुड़े प्रोजेक्ट्स `profileId` के माध्यम से
- कास्केड डिलीट: प्रोफ़ाइल हटाने पर उसके सभी प्रोजेक्ट्स हट जाते हैं

### मल्टी-प्रोवाइडर TTS और कस्टम वॉइस

- **Mistral Voxtral TTS** (डिफ़ॉल्ट): `voxtral-mini-tts-latest`, अतिरिक्त कुंजी की आवश्यकता नहीं
- **ElevenLabs** (वैकल्पिक): `eleven_v3`, नेचुरल वॉयसेज़, इसके लिए `ELEVENLABS_API_KEY` आवश्यक
- प्रदाता एप्लिकेशन सेटिंग्स में कॉन्फ़िगर करने योग्य
- **कस्टम वॉइस**: माता-पिता Mistral Voices API के माध्यम से (ऑडियो सैम्पल से) अपनी वॉइस बना सकते हैं और उन्हें होस्ट/अतिथि भूमिकाओं को असाइन कर सकते हैं — पॉडकास्ट और वॉइस क्विज़ तब माता-पिता की आवाज़ से प्ले होंगे, बच्चे के लिए अनुभव और अधिक इमर्सिव बनाते हैं
- दो वॉयस रोल कॉन्फ़िगर करने योग्य: **होस्ट** (मुख्य नैरेटर) और **अतिथि** (पॉडकास्ट की दूसरी आवाज़)
- सेटिंग्स में Mistral की वॉयस सूची उपलब्ध, भाषा के अनुसार फ़िल्टर करने योग्य

## त्वरित शुरुआत

```bash
# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install

# Configurer les clés API
cp .env.example .env
# Éditez .env avec vos clés :
#   MISTRAL_API_KEY=<your_api_key>           (requis)
#   ELEVENLABS_API_KEY=<your_api_key>        (optionnel, TTS alternatif)
#   SONAR_TOKEN=...                          (optionnel, CI SonarCloud uniquement)

# Lancer le développement
npm run dev
# → Backend :  http://localhost:3000 (API)
# → Frontend : http://localhost:5173 (serveur Vite avec HMR)
```

## कंटेनर के साथ तैनाती

इमेज **GitHub Container Registry** पर प्रकाशित है:

```bash
# Télécharger l'image
podman pull ghcr.io/jls42/eurekai:latest

# Lancer EurekAI
mkdir -p ./data
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=<your_api_key> \
  -e ELEVENLABS_API_KEY=<your_api_key> \
  -v ./data:/app/output:U \
  -p 3000:3000 \
  ghcr.io/jls42/eurekai:latest
# → http://localhost:3000
```

> **`:U`** एक Podman rootless फ्लैग है जो वॉल्यूम पर परमिशन्स को स्वचालित रूप से समायोजित करता है।
> **`ELEVENLABS_API_KEY`** वैकल्पिक है (वैकल्पिक TTS)।

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

| **आयु-आधारित प्रॉम्प्ट्स** | 4 आयु समूह, हर एक के लिए शब्दावली, जटिलता और टोन अलग — एक ही सामग्री सिखाती है अलग तरीके से हर शिक्षार्थी के अनुसार। |
| **एजेंट-आधारित फ़ीचर** | इमेज जेनरेशन और वेब खोज अस्थायी Mistral एजेंट्स का उपयोग करते हैं — स्वच्छ जीवनचक्र और ऑटोमैटिक क्लीनअप। |
| **स्मार्ट URL स्क्रैपिंग** | एक ही फ़ील्ड URLs और कीवर्ड्स मिश्रित स्वीकार करता है — URLs Readability (स्टैटिक पेज) के जरिए स्क्रैप होती हैं, fallback के तौर पर Lightpanda (JS/SPA पेज), जबकि कीवर्ड्स एक Mistral web_search एजेंट ट्रिगर करते हैं। हर परिणाम एक स्वतंत्र स्रोत बनाता है। |
| **मल्टी-प्रोवाइडर TTS** | डिफ़ॉल्ट रूप से Mistral Voxtral TTS (कोई अतिरिक्त की नहीं), वैकल्पिक रूप से ElevenLabs — बिना रिस्टार्ट के कॉन्फ़िगर होने योग्य। |
