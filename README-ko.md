## 이야기 — 왜 EurekAI인가?

**EurekAI**는 [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([공식 사이트](https://worldwide-hackathon.mistral.ai/))(2026년 3월)에서 탄생했습니다. 주제가 필요했고, 아이디어는 아주 현실적인 것에서 나왔습니다. 저는 딸과 함께 정기적으로 시험 대비를 하는데, AI를 활용하면 이를 더 재미있고 상호작용적으로 만들 수 있겠다고 생각했습니다.

목표는 **어떤 입력이든** — 수업 사진, 복사해 붙여넣은 텍스트, 음성 녹음, 웹 검색 — **복습 노트, 플래시카드, 퀴즈, 팟캐스트, 빈칸 채우기, 삽화** 등으로 바꾸는 것입니다. 이 모든 것은 Mistral AI의 프랑스 모델로 구동되며, 프랑스어를 사용하는 학생들에게 자연스럽게 적합한 솔루션이 됩니다.

[초기 프로토타입](https://github.com/jls42/worldwide-hackathon.mistral.ai)은 해커톤 동안 Mistral 서비스에 대한 개념 증명으로 48시간 만에 만들어졌습니다. 이미 작동했지만 제한적이었습니다. 그 이후 EurekAI는 실제 프로젝트가 되었습니다. 빈칸 채우기, 연습문제 탐색, 웹 스크래핑, 설정 가능한 부모 모더레이션, 심층 코드 리뷰 등 훨씬 더 많은 기능이 추가되었습니다. 전체 코드는 AI가 생성했으며 — 주로 [Claude Code](https://code.claude.com/)를 사용했고, 일부는 [Codex](https://openai.com/codex/)와 [Gemini CLI](https://geminicli.com/)를 통해 기여했습니다.

## 자세히 보기 — 기능

### 다중 모달 입력

EurekAI는 프로필에 따라 모더레이션된 4가지 소스 유형을 허용합니다(어린이 및 청소년은 기본적으로 활성화됨):

- **파일 가져오기** — `mistral-ocr-latest`로 처리된 JPG, PNG 또는 PDF 파일(인쇄 텍스트, 표, 손글씨) 또는 직접 가져온 텍스트 파일(TXT, MD).
- **자유 텍스트** — 아무 콘텐츠나 입력하거나 붙여넣으세요. 모더레이션이 활성화된 경우 저장 전에 검토됩니다.
- **음성 입력** — 브라우저에서 오디오를 녹음하세요. `voxtral-mini-latest`가 전사합니다. `language="fr"` 설정은 인식을 최적화합니다.
- **웹 / URL** — 하나 이상의 URL을 붙여넣어 콘텐츠를 직접 스크래핑하거나(JS 페이지는 Readability + Lightpanda 사용), Mistral 에이전트를 통한 웹 검색을 위해 키워드를 입력하세요. 단일 입력 필드는 둘 다 허용하며 — URL과 키워드는 자동으로 분리되고, 각 결과는 독립적인 소스를 생성합니다.

### AI 콘텐츠 생성

생성되는 7가지 학습 자료 유형:

| 생성기 | 모델 | 출력 |
|---|---|---|
| **복습 노트** | `mistral-large-latest` | 제목, 요약, 핵심 포인트, 어휘, 인용구, 일화 |
| **플래시카드** | `mistral-large-latest` | 출처 참조가 포함된 Q/A 카드(개수 설정 가능) |
| **객관식 퀴즈** | `mistral-large-latest` | 객관식 문제, 해설, 적응형 복습(개수 설정 가능) |
| **빈칸 채우기** | `mistral-large-latest` | 힌트가 있는 완성 문장, 관대한 검증(Levenshtein) |
| **팟캐스트** | `mistral-large-latest` + Voxtral TTS | 2인 음성 스크립트 → MP3 오디오 |
| **삽화** | Agent `mistral-large-latest` | 도구 `image_generation`를 통한 교육용 이미지 |
| **음성 퀴즈** | `mistral-large-latest` + Voxtral TTS + STT | TTS 문제 → STT 답변 → AI 검증 |

### 채팅 기반 AI 튜터

수업 문서에 완전히 접근할 수 있는 대화형 튜터:

- `mistral-large-latest` 사용
- **도구 호출**: 대화 중에 노트, 플래시카드, 퀴즈 또는 빈칸 채우기를 생성할 수 있음
- 강좌당 50개 메시지 기록
- 프로필에 대해 활성화된 경우 콘텐츠 모더레이션

### 자동 라우터

라우터는 `mistral-small-latest`를 사용해 소스 콘텐츠를 분석하고, 사용 가능한 7개 생성기 중 가장 적합한 것을 제안합니다. 인터페이스는 실시간 진행률을 표시합니다. 먼저 분석 단계가 있고, 그 다음 개별 생성이 진행되며 취소할 수 있습니다.

### 적응형 학습

- **퀴즈 통계**: 문제별 시도 횟수와 정확도 추적
- **퀴즈 복습**: 약한 개념을 겨냥한 5~10개의 새 문제 생성
- **지시문 감지**: 복습 지시문("나는 ...을 알면 내 수업을 안다")을 감지하고 호환되는 텍스트 생성기(노트, 플래시카드, 퀴즈, 빈칸 채우기)에서 우선 적용

### 보안 및 자녀 보호

- **4개 연령 그룹**: 어린이(≤10세), 청소년(11-15), 학생(16-25), 성인(26+)
- **콘텐츠 모더레이션**: `mistral-moderation-latest` 및 10개 사용 가능한 범주, 어린이/청소년에 대해 기본적으로 5개 차단 (`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`). 범주는 설정에서 프로필별로 사용자 지정 가능.
- **부모 PIN**: SHA-256 해시, 15세 미만 프로필에 필요. 프로덕션 배포에서는 솔트가 있는 느린 해시(Argon2id, bcrypt)를 사용할 것.
- **채팅 제한**: 16세 미만은 AI 채팅이 기본적으로 비활성화되어 있으며, 부모가 활성화 가능

### 다중 프로필 시스템

- 이름, 나이, 아바타, 언어 기본 설정이 있는 여러 프로필
- `profileId`를 통해 프로필에 연결된 프로젝트
- 연쇄 삭제: 프로필을 삭제하면 해당 프로필의 모든 프로젝트가 삭제됨

### 다중 제공자 TTS 및 사용자 지정 음성

- **Mistral Voxtral TTS**(기본): `voxtral-mini-tts-latest`, 추가 키 필요 없음
- **ElevenLabs**(대안): `eleven_v3`, 자연스러운 음성, `ELEVENLABS_API_KEY` 필요
- 앱 설정에서 제공자 구성 가능
- **사용자 지정 음성**: 부모는 Mistral Voices API를 통해(오디오 샘플로부터) 자신만의 음성을 만들고 호스트/게스트 역할에 할당할 수 있습니다. 그러면 팟캐스트와 음성 퀴즈가 부모의 음성으로 읽혀져 아이에게 더욱 몰입감 있는 경험을 제공합니다
- 구성 가능한 두 음성 역할: **호스트**(주 내레이터)와 **게스트**(팟캐스트의 두 번째 음성)
- Mistral 음성 전체 카탈로그는 설정에서 사용할 수 있으며 언어별로 필터링 가능

### 국제화

- 인터페이스는 9개 언어로 제공: fr, en, es, pt, it, nl, de, hi, ar
- AI 프롬프트는 15개 언어 지원: fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv
- 프로필별 언어 설정 가능

| **TTS(기본)** | Mistral Voxtral TTS | `voxtral-mini-tts-latest`, 내장 음성 합성 |

| **Markdown** | Marked | 채팅에서 마크다운 렌더링 |
| **파일 업로드** | Multer 2.x | multipart 폼 처리 |
| **오디오** | ffmpeg-static | 오디오 세그먼트 결합 |
| **테스트** | Vitest | 단위 테스트 — 커버리지는 SonarCloud로 측정 |
| **저장소** | JSON 파일 | 의존성 없는 저장 |

```bash
# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install
```

```bash
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=<your_api_key> \
```

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

## 아키텍처 결정

| 결정 | 근거 |
|---|---|
| **React/Vue 대신 Alpine.js** | 최소한의 용량, Vite로 컴파일된 TypeScript와 함께 가벼운 반응성. 속도가 중요한 해커톤에 완벽함. |
| **JSON 파일 기반 영속성** | 의존성 없음, 즉시 시작. 구성해야 할 데이터베이스가 없음 — 시작하고 바로 진행 가능. |
| **Vite + Handlebars** | 두 세계의 장점: 개발용 빠른 HMR, 코드 구성을 위한 HTML partials, Tailwind JIT. |
| **중앙화된 프롬프트** | 모든 AI 프롬프트를 `prompts.ts`에 — 언어/연령대별로 쉽게 반복, 테스트, 조정 가능. |
| **다중 생성 시스템** | 각 생성은 자체 ID를 가진 독립 객체 — 하나의 과정에 여러 시트, 퀴즈 등을 허용. |
| **연령별 맞춤 프롬프트** | 4개 연령 그룹별로 어휘, 복잡성, 톤이 다름 — 같은 콘텐츠도 학습자에 따라 다르게 가르침. |
| **Agents 기반 기능** | 이미지 생성과 웹 검색은 임시 Mistral Agents를 사용 — 자동 정리되는 깔끔한 생명주기. |
| **지능형 URL 스크래핑** | 하나의 입력란에서 URL과 키워드를 혼합해 받음 — URL은 Readability(정적 페이지)로 스크랩하고 Lightpanda(JS/SPA 페이지)로 폴백, 키워드는 Mistral web_search Agent를 트리거함. 각 결과는 독립적인 소스를 생성함. |
| **다중 제공자 TTS** | 기본은 Mistral Voxtral TTS(추가 키 불필요), 대안으로 ElevenLabs — 재시작 없이 설정 가능. |
