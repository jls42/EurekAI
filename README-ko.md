## 이야기 — 왜 EurekAI인가?

**EurekAI**는 [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([공식 사이트](https://worldwide-hackathon.mistral.ai/))에서 태어났습니다 (2026년 3월). 주제가 필요했는데 — 아이디어는 아주 실용적인 필요에서 왔습니다: 저는 딸과 함께 정기적으로 시험 준비를 하는데, 인공지능으로 이를 더 재미있고 대화형으로 만들 수 있겠다고 생각했습니다.

목표는: **어떤 입력이든** — 수업 사진, 복사·붙여넣은 텍스트, 음성 녹음, 웹 검색 — 을 받아 **요약 노트, 플래시카드, 퀴즈, 팟캐스트, 빈칸 채우기 문제, 삽화 등** 으로 변환하는 것입니다. 모든 것은 프랑스 기반 Mistral AI 모델로 구동되어 프랑스어 사용자에게 자연스럽게 적합합니다.

[초기 프로토타입](https://github.com/jls42/worldwide-hackathon.mistral.ai)은 해커톤 동안 48시간 내에 Mistral 서비스들을 중심으로 개념 증명으로 제작되었습니다 — 이미 동작하지만 제한적이었습니다. 이후 EurekAI는 진짜 프로젝트가 되었고: 빈칸 채우기, 연습 문제 내비게이션, 웹 스크래핑, 구성 가능한 부모용 검열, 심층 코드 리뷰 등 많은 기능이 추가되었습니다. 전체 코드의 대부분은 IA로 생성되었으며 — 주로 [Claude Code](https://code.claude.com/)를 사용하고 일부는 [Codex](https://openai.com/codex/)와 [Gemini CLI](https://geminicli.com/)의 기여가 있습니다.

## 심층 기능 설명

### 멀티모달 입력

EurekAI는 프로필별(기본적으로 아동 및 청소년에 대해 활성화)로 검열되는 4가지 유형의 소스를 허용합니다:

- **파일 가져오기** — JPG, PNG 또는 PDF 파일은 `mistral-ocr-latest`로 처리됩니다 (인쇄된 텍스트, 표, 손글씨 포함). 또는 텍스트 파일(TXT, MD)은 직접 가져오기 가능합니다.
- **자유 텍스트** — 어떤 콘텐츠든 입력하거나 붙여넣기. 저장 전 검열이 활성화되어 있으면 검열됩니다.
- **음성 입력** — 브라우저에서 오디오를 녹음. `voxtral-mini-latest`가 전사합니다. `language="fr"` 설정은 인식률을 최적화합니다.
- **웹 / URL** — 하나 이상의 URL을 붙여넣어 직접 스크래핑(Readability + Lightpanda로 JS 페이지 처리)하거나 키워드를 입력하여 Agent Mistral를 통한 웹 검색 수행. 단일 입력 필드는 URL 및 키워드를 모두 허용하며 자동으로 분리하고 각 결과는 독립적인 소스로 생성됩니다.

### IA 기반 콘텐츠 생성

생성되는 학습 자료 유형은 7가지입니다:

| 생성기 | 모델 | 출력 |
|---|---|---|
| **요약 노트** | `mistral-large-latest` | 제목, 요약, 핵심 포인트, 어휘, 인용문, 일화 |
| **플래시카드** | `mistral-large-latest` | 출처 참조가 포함된 Q/A 카드 (수량 설정 가능) |
| **객관식 퀴즈** | `mistral-large-latest` | 객관식 문제, 해설, 적응형 복습 (수량 설정 가능) |
| **빈칸 채우기** | `mistral-large-latest` | 힌트가 있는 완성 문장, 관용적 검증(Levenshtein) |
| **팟캐스트** | `mistral-large-latest` + Voxtral TTS | 2인 스크립트 → MP3 오디오 |
| **삽화** | Agent `mistral-large-latest` | 도구 `image_generation`을 통한 교육용 이미지 |
| **음성 퀴즈** | `mistral-large-latest` + Voxtral TTS + STT | TTS로 질문 → STT로 답변 → IA 검증 |

### 채팅 기반 AI 튜터

수업 문서에 완전히 접근 가능한 대화형 튜터:

- `mistral-large-latest` 사용
- **도구 호출**: 대화 중에 노트, 플래시카드, 퀴즈 또는 빈칸 채우기 생성 가능
- 과목별로 50개 메시지의 히스토리 유지
- 프로필에 대해 검열이 활성화되어 있으면 콘텐츠가 필터링됨

### 자동 라우터

라우터는 소스의 콘텐츠를 분석하고 7가지 생성기 중 가장 적절한 것을 제안하기 위해 `mistral-small-latest`를 사용합니다. UI는 실시간 진행 상황을 표시합니다: 먼저 분석 단계, 그다음 개별 생성이 진행되며 취소 가능.

### 적응 학습

- **퀴즈 통계**: 문제별 시도 및 정답률 추적
- **퀴즈 복습**: 약한 개념을 겨냥한 5~10개의 새 문제 생성
- **지시 감지**: 복습 지시("나는 ...을 알면 내 수업을 안다" 등)를 감지하고 호환 가능한 텍스트 생성기(요약 노트, 플래시카드, 퀴즈, 빈칸)에 우선순위 부여

### 보안 및 부모 통제

- **4개 연령 그룹**: 아동(≤10세), 청소년(11-15세), 학생(16-25세), 성인(26세+)
- **콘텐츠 검열**: `mistral-moderation-latest` 사용, 10개 카테고리 제공, 아동/청소년에 대해 기본적으로 5개 차단(`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`). 프로필 설정에서 카테고리 맞춤 가능.
- **부모 PIN**: SHA-256 해시, 15세 미만 프로필에 필요. 프로덕션 배포 시에는 솔트가 있는 느린 해시(Argon2id, bcrypt 등)를 사용 권장.
- **채팅 제한**: 16세 미만은 기본적으로 AI 채팅 비활성화, 부모가 활성화 가능

### 다중 프로필 시스템

- 이름, 나이, 아바타, 언어 선호를 가진 다중 프로필 지원
- 프로젝트는 프로필과 연동 (`profileId`)
- 계정 삭제 시 연쇄 삭제: 프로필 삭제하면 해당 프로필의 모든 프로젝트 삭제

### 다중 제공자 TTS 및 맞춤 음성

- **Mistral Voxtral TTS** (기본): `voxtral-mini-tts-latest`, 추가 키 불필요
- **ElevenLabs** (대안): `eleven_v3`, 자연스러운 음성, `ELEVENLABS_API_KEY` 필요
- 제공자는 앱 설정에서 구성 가능
- **맞춤 음성**: 부모는 Mistral Voices API를 통해 오디오 샘플로 자신의 음성을 생성하고 호스트/게스트 역할에 할당할 수 있으며 — 팟캐스트와 음성 퀴즈는 부모 음성으로 재생되어 아이의 몰입감을 높임
- 두 개의 음성 역할 설정 가능: **호스트**(주 내레이터) 및 **게스트**(팟캐스트의 두 번째 음성)
- 설정에서 언어별로 필터링 가능한 전체 Mistral 음성 카탈로그 제공

### 국제화

- UI는 9개 언어로 제공: fr, en, es, pt, it, nl, de, hi, ar
- AI 프롬프트는 15개 언어 지원(fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv)
- 프로필별 언어 설정 가능

| **파일 업로드** | Multer 2.x | multipart 폼 처리 |
| **오디오** | ffmpeg-static | 오디오 세그먼트 연결 |
| **테스트** | Vitest | 단위 테스트 — SonarCloud로 커버리지 측정 |

## 컨테이너로 배포

이미지는 **GitHub Container Registry**에 게시되어 있습니다:

```bash
# Télécharger l'image
podman pull ghcr.io/jls42/eurekai:latest

# Lancer EurekAI
mkdir -p ./data
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=<your_api_key> \
  -e ELEVENLABS_API_KEY=<your_api_key> \
  -v ./data:/app/output:U \
  -p 3000:3000 \
  ghcr.io/jls42/eurekai:latest
# → http://localhost:3000
```

> **`:U`**는 Podman rootless 플래그로 볼륨 권한을 자동으로 조정합니다.
> **`ELEVENLABS_API_KEY`**는 선택 사항입니다 (대체 TTS).

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

## 아키텍처 결정

| 결정 | 근거 |
|---|---|
| **React/Vue 대신 Alpine.js** | 최소한의 오버헤드, Vite로 컴파일된 TypeScript로 가벼운 반응성. 속도가 중요한 해커톤에 적합. |
| **JSON 파일로 영속화** | 종속성 없음, 즉시 시작. 데이터베이스 설정 불필요 — 바로 시작 가능. |
| **Vite + Handlebars** | 양쪽의 장점: 개발을 위한 빠른 HMR, 코드 구성을 위한 HTML partials, Tailwind JIT. |
| **중앙집중형 프롬프트** | 모든 AI 프롬프트는 `prompts.ts` 에 있음 — 언어/연령대별로 반복, 테스트, 조정하기 쉬움. |
| **다중 생성 시스템** | 각 생성은 고유 ID를 가진 독립 객체 — 코스별로 여러 개의 노트, 퀴즈 등을 허용. |
| **연령별 맞춤 프롬프트** | 어휘, 난이도, 어조가 다른 4개 연령대 그룹 — 동일한 내용이 학습자에 따라 다르게 가르쳐짐. |
| **에이전트 기반 기능** | 이미지 생성 및 웹 검색은 임시 Mistral 에이전트를 사용 — 자체 수명 주기 및 자동 정리. |
| **스마트 URL 스크래핑** | 하나의 입력 필드에 URL과 키워드를 섞어 입력 가능 — URL은 Readability(정적 페이지)를 통해 스크래핑하고, JS/SPA 페이지에는 Lightpanda를 폴백으로 사용, 키워드는 Mistral 에이전트의 web_search를 트리거. 각 결과는 독립된 소스를 생성. |
| **다중 TTS 제공자** | 기본은 Mistral Voxtral TTS(추가 키 불필요), 대안으로 ElevenLabs — 재시작 없이 구성 가능. |
