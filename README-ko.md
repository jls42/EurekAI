## 이야기 — 왜 EurekAI인가?

**EurekAI**는 [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([공식 사이트](https://worldwide-hackathon.mistral.ai/))(2026년 3월)에서 탄생했습니다. 주제가 필요했는데, 아주 현실적인 아이디어에서 출발했습니다. 저는 딸과 함께 정기적으로 시험 공부를 준비하는데, AI를 활용하면 이를 더 재미있고 인터랙티브하게 만들 수 있지 않을까 생각했습니다.

목표는 **어떤 입력이든** — 수업 사진, 복사해 붙여넣은 텍스트, 음성 녹음, 웹 검색 — 받아서 **복습 카드, 플래시카드, 퀴즈, 팟캐스트, 빈칸 채우기, 일러스트 등**으로 변환하는 것입니다. 모든 것은 Mistral AI의 프랑스 모델로 구동되며, 프랑스어를 사용하는 학생들에게 자연스럽게 적합한 솔루션입니다.

[초기 프로토타입](https://github.com/jls42/worldwide-hackathon.mistral.ai)은 해커톤 동안 48시간 만에 Mistral 서비스 기반의 개념 증명을 위해 만들어졌으며, 이미 동작했지만 제한적이었습니다. 그 이후 EurekAI는 진짜 프로젝트가 되었습니다: 빈칸 채우기, 연습 문제 탐색, 웹 스크래핑, 설정 가능한 부모용 모더레이션, 심층 코드 리뷰 등 훨씬 더 많은 기능을 갖추게 되었습니다. 전체 코드는 AI로 생성되며 — 주로 [Claude Code](https://code.claude.com/)를 사용했고, [Codex](https://openai.com/codex/)와 [Gemini CLI](https://geminicli.com/)를 통한 일부 기여도 있습니다.

## 심층 살펴보기 — 기능

### 멀티모달 입력

EurekAI는 프로필에 따라 모더레이션되는 4가지 소스 유형을 허용합니다(어린이와 청소년은 기본 활성화) :

- **파일 가져오기** — JPG, PNG 또는 PDF 파일은 `mistral-ocr-latest`(인쇄 텍스트, 표, 손글씨)로 처리되며, 텍스트 파일(TXT, MD)은 직접 가져옵니다. 다중 파일 업로드는 **업로드 세션** 시스템을 사용합니다: 파일별 개별 진행률, 실패한 파일만 재시도, 완료 시 세션 닫기. OCR은 페이지별 **신뢰도 점수**(`overall` + `perPage[]`)를 제공하며, UI에서는 tier `high` / `medium` / `low` 배지 형태로 표시됩니다(임계값 약 0.9 / 약 0.7) — 품질이 나쁜 스캔이라도 차단하지 않고 경고합니다.
- **자유 텍스트** — 아무 콘텐츠나 입력하거나 붙여넣으세요. 모더레이션이 활성화되어 있으면 저장 전에 검토됩니다.
- **음성 입력** — 브라우저에서 오디오를 녹음합니다. `voxtral-mini-latest`가 이를 전사합니다. `language="fr"` 파라미터는 인식을 최적화합니다.
- **웹 / URL** — 하나 이상의 URL을 붙여넣어 콘텐츠를 직접 스크래핑하거나(Readability + Lightpanda는 JS 페이지용), 키워드를 입력하여 Mistral 에이전트를 통한 웹 검색을 수행합니다. 단일 필드는 둘 다 허용하며 — URL과 키워드는 자동으로 분리되고, 각 결과는 독립적인 소스를 생성합니다.

### AI 콘텐츠 생성

생성되는 7가지 학습 자료 유형:

| 생성기 | 모델 | 출력 |
|---|---|---|
| **복습 카드** | `mistral-large-latest` | 제목, 요약, 핵심 포인트, 어휘, 인용문, 일화 |
| **플래시카드** | `mistral-large-latest` | 소스 참조가 포함된 Q/A 카드(개수 설정 가능) |
| **객관식 퀴즈** | `mistral-large-latest` | 객관식 문제, 설명, 적응형 복습(개수 설정 가능) |
| **빈칸 채우기** | `mistral-large-latest` | 힌트와 관대한 검증(Levenshtein)이 있는 완성 문장 |
| **팟캐스트** | `mistral-large-latest` + Voxtral TTS | 2인 스크립트 → MP3 오디오 |
| **일러스트** | 에이전트 `mistral-large-latest` | `image_generation` 도구를 통한 교육용 이미지 |
| **음성 퀴즈** | `mistral-large-latest` + Voxtral TTS + STT | TTS 질문 → STT 응답 → AI 검증 |

### 채팅 기반 AI 튜터

수업 문서에 완전히 접근할 수 있는 대화형 튜터:

- `mistral-large-latest` 사용
- **도구 호출**: 대화 중에 복습 카드, 플래시카드, 퀴즈 또는 빈칸 채우기를 생성할 수 있음
- 수업당 50개 메시지 기록
- 프로필에 활성화되어 있으면 콘텐츠 모더레이션

### 자동 라우터

라우터는 `mistral-small-latest`를 사용해 소스 콘텐츠를 분석하고, 사용 가능한 7가지 생성기 중 가장 관련성 높은 것을 제안합니다. 인터페이스는 진행 상황을 실시간으로 표시합니다: 먼저 분석 단계, سپس 개별 생성 단계와 취소 가능 옵션이 이어집니다.

### 적응형 학습

- **퀴즈 통계**: 문항별 시도 횟수와 정답률 추적
- **퀴즈 복습**: 약한 개념을 대상으로 5~10개의 새 문제 생성
- **지시문 감지**: 복습 지시문("Je sais ma leçon si je sais...")을 감지하고 호환되는 텍스트 생성기(복습 카드, 플래시카드, 퀴즈, 빈칸 채우기)에서 우선 적용

### 보안 및 부모 통제

- **4개 연령 그룹**: 어린이(≤10세), 청소년(11-15), 학생(16-25), 성인(26+)
- **콘텐츠 모더레이션**: 사용 가능한 10개 카테고리의 `mistral-moderation-latest`, 그중 5개는 어린이/청소년에게 기본 차단(`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`). 카테고리는 설정에서 프로필별로 사용자 지정 가능.
- **부모 PIN**: SHA-256 해시, 15세 미만 프로필에 필요합니다. 운영 환경 배포 시에는 솔트가 있는 느린 해시(Argon2id, bcrypt)를 사용하세요.
- **채팅 제한**: 16세 미만은 AI 채팅이 기본 비활성화되며, 부모가 활성화할 수 있음

### 다중 프로필 시스템

- 이름, 나이, 아바타, 언어 선호도를 가진 여러 프로필
- **프로필별 음성**(`Profile.mistralVoices?: { host, guest }`) — 각 어린이는 팟캐스트/음성 퀴즈용 음성 쌍을 가질 수 있음
- **프로필별 테마**(`Profile.theme: 'dark' | 'light'`) — 프로필 변경 시 자동 전환, 백엔드에 유지됨
- 프로필과 연결된 프로젝트는 `profileId`를 통해 관리
- 계단식 삭제: 프로필을 삭제하면 해당 프로필의 모든 프로젝트가 삭제됨

### API 비용 추적

모든 Mistral 호출(chat, OCR, STT, TTS, 모더레이션, 에이전트)은 사용자에게 **투명한** € 추정치를 제공하도록 계측되어 있습니다 — 청구에 대한 놀라움이 없습니다.

- **기준 소스**: `helpers/pricing.ts` — 모델 prefix별 `MODEL_PRICING`(예: `mistral-large` → input 0.5 €/M tokens, output 1.5 €/M tokens), 주기적 재스크래핑을 위한 Mistral 문서 URL이 포함된 `PRICING_SOURCES`
- **지원 단위**: `tokens`, `characters`(TTS), `pages`(OCR), `audio-seconds`(STT) — `helpers/cost-calc.ts`가 변환을 제어
- **계측 체인**: `helpers/tracked-client.ts`(Mistral 클라이언트 래퍼) → `helpers/usage-context.ts`(AsyncLocalStorage) → `helpers/cost-calc.ts` → `helpers/cost-persist.ts` → `helpers/cost-middleware.ts`(HTTP 응답에 주입)
- **UI**: 생성별 비용 배지(`src/partials/cost-badge-gen.html`), 소스별(`cost-badge-src.html`), 대시보드의 누적 합계(`Project.totalCost`)
- **엔드포인트**: `/generate/*` 및 `/sources/*` 응답에 `costDelta: { estimatedEuros, perModel }`가 포함됩니다. `GET /projects/:pid`는 `totalCost` + `costLog[]` 기록을 반환합니다. `GET /api/config/status`는 `costEstimateAvailable`을 노출합니다

### 다중 제공자 TTS 및 사용자 지정 음성

- **Mistral Voxtral TTS**(기본) : `voxtral-mini-tts-latest`, 추가 키 필요 없음
- **ElevenLabs**(대안) : `eleven_v3`, 자연스러운 음성, `ELEVENLABS_API_KEY` 필요
- 앱 설정에서 제공자 구성 가능
- **사용자 지정 음성**: 부모는 Mistral Voices API를 통해(오디오 샘플 기반) 자신만의 음성을 만들고 호스트/게스트 역할에 할당할 수 있으며 — 그러면 팟캐스트와 음성 퀴즈가 부모의 목소리로 재생되어 아이에게 더욱 몰입감 있는 경험을 제공합니다
- 설정 가능한 두 음성 역할: **호스트**(주요 내레이터) 및 **게스트**(팟캐스트의 두 번째 목소리)
- Mistral 음성 전체 카탈로그는 설정에서 제공되며, 언어별 필터링 가능

### 국제화

- 인터페이스는 9개 언어로 제공됨: fr, en, es, pt, it, nl, de, hi, ar
- AI 프롬프트는 15개 언어를 지원함(fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv)
- 프로필별 언어 설정 가능

# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install

# Configurer les clés API
cp .env.example .env
# Éditez .env avec vos clés :
#   MISTRAL_API_KEY=<your_api_key>           (requis)
#   ELEVENLABS_API_KEY=<your_api_key>        (optionnel, TTS alternatif)

npm run dev

  -e MISTRAL_API_KEY=<your_api_key> \

## 아키텍처 결정

| 결정 | 근거 |
|---|---|
| **React/Vue 대신 Alpine.js** | 최소한의 크기, Vite로 컴파일된 TypeScript와 함께 가벼운 반응성. 속도가 중요한 해커톤에 적합합니다. |
| **JSON 파일 기반 영속성** | 의존성 없음, 즉시 시작. 설정할 데이터베이스가 없습니다 — 시작하면 바로 사용 가능합니다. |
| **Vite + Handlebars** | 개발용 빠른 HMR, 코드 구성을 위한 HTML partials, Tailwind JIT — 두 세계의 장점만 결합. |
| **중앙 집중형 프롬프트** | 모든 AI 프롬프트는 `prompts.ts`에 있음 — 언어/연령대별로 쉽게 반복, 테스트, 조정 가능. |
| **다중 생성 시스템** | 각 생성은 고유 ID를 가진 독립 객체 — 한 강의당 여러 학습 카드, 퀴즈 등을 허용. |
| **연령 맞춤형 프롬프트** | 어휘, 복잡성, 어조가 다른 4개 연령 그룹 — 같은 콘텐츠도 학습자에 따라 다르게 가르칩니다. |
| **에이전트 기반 기능** | 이미지 생성과 웹 검색은 임시 Mistral Agent를 사용 — 자동 정리를 포함한 깔끔한 생명주기. |
| **지능형 URL 스크래핑** | 단일 필드에서 URL과 키워드를 혼합해 입력 가능 — URL은 Readability(정적 페이지)로 스크랩하고 Lightpanda(JS/SPA 페이지)로 폴백하며, 키워드는 Mistral web_search Agent를 트리거합니다. 각 결과는 독립적인 소스를 생성합니다. |
| **멀티 provider TTS** | 기본은 Mistral Voxtral TTS(추가 키 불필요), 대안은 ElevenLabs — 재시작 없이 설정 가능. |
