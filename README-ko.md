## 이야기 — 왜 EurekAI인가 ?

**EurekAI**는 [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([공식 사이트](https://worldwide-hackathon.mistral.ai/))(2026년 3월) 동안 탄생했습니다. 주제가 필요했는데, 아이디어는 매우 현실적인 상황에서 나왔습니다: 저는 딸과 함께 정기적으로 시험을 준비하는데, AI를 통해 이를 더 재미있고 상호작용적으로 만들 수 있겠다고 생각했습니다.

목표는: **어떤 입력이든** — 수업 사진, 복사해 온 텍스트, 음성 녹음, 웹 검색 — 을 받아서 **요약 노트, 플래시카드, 퀴즈, 팟캐스트, 빈칸 채우기, 일러스트 등**으로 변환하는 것입니다. 모든 것은 Mistral AI의 프랑스 모델로 구동되어 프랑스어권 학생들에게 자연스럽게 적합한 솔루션이 됩니다.

[초기 프로토타입](https://github.com/jls42/worldwide-hackathon.mistral.ai)은 해커톤 동안 48시간 만에 Mistral 서비스들을 중심으로 개념 증명으로 제작되었습니다 — 이미 작동하지만 제한적이었습니다. 그 이후 EurekAI는 실제 프로젝트로 발전했습니다: 빈칸 채우기, 연습 문제 내비게이션, 웹 스크레이핑, 프로필별 구성 가능한 부모용 검열, 심층 코드 리뷰 등. 전체 코드의 대부분은 AI에 의해 생성되었으며 — 주로 [Claude Code](https://code.claude.com/)를 사용했고 일부는 [Codex](https://openai.com/codex/)와 [Gemini CLI](https://geminicli.com/)로 기여했습니다.

## 심층 분석 — 기능

### 다중 모달 입력

EurekAI는 프로필에 따라(기본적으로 어린이 및 청소년에 대해 활성화) 4가지 유형의 소스를 허용합니다:

- **파일 가져오기** — JPG, PNG 또는 PDF 파일은 `mistral-ocr-latest`로 처리(인쇄된 텍스트, 표, 손글씨), 또는 TXT/MD 같은 텍스트 파일은 직접 가져오기.
- **텍스트 입력** — 텍스트를 직접 입력하거나 붙여넣기. 저장 전에 검열이 활성화되어 있으면 검열 처리.
- **음성 입력** — 브라우저에서 오디오 녹음. `voxtral-mini-latest`로 전사. `language="fr"` 파라미터가 인식 성능을 최적화.
- **웹 / URL** — 하나 이상의 URL을 붙여넣어 콘텐츠를 직접 스크레이핑(Readability + Lightpanda는 JS 페이지용), 또는 키워드를 입력해 Agent Mistral로 웹 검색. 단일 입력란은 URL과 키워드 둘 다 허용 — 자동으로 분리되며 각 결과는 독립 소스로 생성.

### AI 콘텐츠 생성

생성 가능한 학습 자료는 7가지 유형:

| 생성기 | 모델 | 출력 |
|---|---|---|
| **요약 노트** | `mistral-large-latest` | 제목, 요약, 핵심 포인트, 어휘, 인용구, 일화 |
| **플래시카드** | `mistral-large-latest` | 출처 참조가 포함된 Q/A 카드(개수 조정 가능) |
| **객관식 퀴즈** | `mistral-large-latest` | 선택형 질문, 해설, 적응형 복습(개수 조정 가능) |
| **빈칸 채우기** | `mistral-large-latest` | 힌트가 있는 완성형 문장, 관용적(Levenshtein) 검증 |
| **팟캐스트** | `mistral-large-latest` + Voxtral TTS | 2인 스크립트 → MP3 오디오 |
| **일러스트** | Agent `mistral-large-latest` | `image_generation` 도구로 생성된 교육용 이미지 |
| **음성 퀴즈** | `mistral-large-latest` + Voxtral TTS + STT | 질문 TTS → 응답 STT → AI 검증 |

### 챗 기반 AI 튜터

수업 문서 전체에 접근 가능한 대화형 튜터:

- `mistral-large-latest` 사용
- **도구 호출**: 대화 중에 요약 노트, 플래시카드, 퀴즈 또는 빈칸 채우기를 생성 가능
- 과정별로 50개 메시지의 히스토리 보관
- 프로필에서 검열이 활성화되어 있으면 콘텐츠 검열 적용

### 자동 라우터

라우터는 콘텐츠를 분석하고 7가지 생성기 중 가장 적절한 생성기를 제안하기 위해 `mistral-small-latest`을 사용합니다. UI는 실시간 진행 상황을 표시: 먼저 분석 단계, 그다음 각 생성의 개별 진행과 취소 가능 옵션.

### 적응형 학습

- **퀴즈 통계**: 문제별 시도 횟수 및 정답률 추적
- **퀴즈 복습**: 취약 개념을 겨냥한 5–10개의 신규 문제 생성
- **지시 감지**: "나는 ...을 알면 시험을 잘 본다" 같은 복습 지시를 감지하여 호환되는 텍스트 생성기(요약 노트, 플래시카드, 퀴즈, 빈칸 채우기)에 우선 반영

### 보안 및 부모 통제

- **4개 연령 그룹**: 어린이 (≤10세), 청소년 (11–15), 학생 (16–25), 성인 (26+)
- **콘텐츠 검열**: `mistral-moderation-latest` 사용, 10개 카테고리 제공, 어린이/청소년 기본으로 5개 차단(`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`). 프로필 설정에서 카테고리 사용자화 가능.
- **부모 PIN**: SHA-256 해시, 15세 미만 프로필에 필수. 운영 환경에서는 솔트가 있는 느린 해시(Argon2id, bcrypt) 사용 권장.
- **채팅 제한**: 16세 미만은 기본적으로 AI 채팅 비활성화, 부모가 활성화 가능

### 다중 프로필 시스템

- 이름, 나이, 아바타, 언어 선호를 가진 다중 프로필 지원
- 프로필에 연결된 프로젝트는 `profileId` 통해 관리
- 연쇄 삭제: 프로필 삭제 시 해당 프로필의 모든 프로젝트 삭제

### 다중 TTS 제공자 및 맞춤 음성

- **Mistral Voxtral TTS** (기본): `voxtral-mini-tts-latest`, 추가 키 불필요
- **ElevenLabs** (대안): `eleven_v3`, 자연스러운 음성, `ELEVENLABS_API_KEY` 필요
- 애플리케이션 설정에서 제공자 선택 가능
- **맞춤 음성**: 부모는 Mistral Voices API로 음성 샘플을 통해 자신의 음성을 생성하고 호스트/게스트 역할에 할당 가능 — 팟캐스트와 음성 퀴즈가 부모의 목소리로 재생되어 아이에게 더 몰입감 있는 경험 제공
- 설정 가능한 두 개의 음성 역할: **호스트**(주해설)와 **게스트**(두 번째 음성)
- 언어별 필터링 가능한 Mistral 전체 음성 카탈로그 제공

### 다국어 지원

- 인터페이스는 9개 언어로 제공: fr, en, es, pt, it, nl, de, hi, ar
- AI 프롬프트는 15개 언어를 지원: fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv
- 프로필별 언어 설정 가능

| **마크다운** | Marked | 채팅 내 마크다운 렌더링 |
| **파일 업로드** | Multer 2.x | multipart 폼 처리 |
| **오디오** | ffmpeg-static | 오디오 세그먼트 합치기 |

| `mistral-moderation-latest` | 콘텐츠 검열 | 어린이/청소년용 5개 카테고리 차단 (+ 탈옥 탐지) |

## 아키텍처 결정

| 결정 | 정당성 |
|---|---|
| **React/Vue 대신 Alpine.js** | 최소한의 오버헤드, Vite로 컴파일된 TypeScript와 함께 가벼운 반응성. 속도가 중요한 해커톤에 적합합니다. |
| **JSON 파일로 영속성 유지** | 의존성 제로, 즉시 시작 가능. 데이터베이스 설정 필요 없음 — 바로 시작할 수 있습니다. |
| **Vite + Handlebars** | 두 기술의 장점: 빠른 HMR, 코드 정리를 위한 HTML partials, Tailwind JIT. |
| **중앙화된 프롬프트** | 모든 AI 프롬프트를 `prompts.ts`에 모아둠 — 언어/연령대별로 반복 테스트 및 조정이 용이합니다. |
| **다중 생성 시스템** | 각 생성은 고유 ID를 가진 독립 객체 — 코스별로 여러 복습지, 퀴즈 등을 허용합니다. |
| **연령대별로 맞춘 프롬프트** | 어휘, 복잡도, 톤이 다른 4개 연령대 그룹 — 같은 콘텐츠라도 학습자에 따라 다르게 가르칩니다. |
| **에이전트 기반 기능들** | 이미지 생성 및 웹 검색은 임시 Mistral 에이전트를 사용 — 수명 주기 관리와 자동 정리 제공. |
| **지능형 URL 스크래핑** | 하나의 필드에 URL과 키워드를 혼합 입력 가능 — URL은 Readability(정적 페이지)로 스크랩, JS/SPA는 Lightpanda로 폴백, 키워드는 Mistral web_search 에이전트 트리거. 각 결과는 독립 소스로 생성됩니다. |
| **다중 TTS 제공자** | 기본은 Mistral Voxtral TTS(추가 키 불필요), 대안으로 ElevenLabs — 재시작 없이 구성 가능. |
