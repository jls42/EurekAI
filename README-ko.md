## 배경 — 왜 EurekAI인가?

**EurekAI**는 [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([공식 사이트](https://worldwide-hackathon.mistral.ai/))(2026년 3월) 동안 탄생했습니다. 저는 주제가 필요했고, 아이와 숙제나 시험 대비를 자주 준비하면서 AI로 이를 더 재미있고 상호작용적으로 만들 수 있겠다고 생각하게 되었습니다.

목표는: **어떤 입력이든** — 수업 사진, 복사-붙여넣은 텍스트, 음성 녹음, 웹 검색 — 을 받아 **복습 노트, 플래시카드, 퀴즈, 팟캐스트, 빈칸 문제, 일러스트레이션 등**으로 변환하는 것입니다. 모든 처리에 프랑스의 Mistral AI 모델을 활용해 프랑스어 사용자에게 자연스럽게 맞춰진 솔루션을 제공합니다.

초기 [프로토타입](https://github.com/jls42/worldwide-hackathon.mistral.ai)은 해커톤 기간 48시간 동안 Mistral 서비스 기반의 개념 증명으로 만들어졌습니다 — 이미 동작하지만 제한적이었습니다. 그 이후로 EurekAI는 실제 프로젝트가 되었습니다: 빈칸 문제, 연습 내비게이션, 웹 스크래핑, 설정 가능한 부모용 검열, 상세 코드 리뷰 등으로 발전했습니다. 전체 코드베이스는 주로 [Claude Code](https://code.claude.com/)로 생성되었고, 일부는 [Codex](https://openai.com/codex/)와 [Gemini CLI](https://geminicli.com/)의 기여가 있습니다.

## 심층 분석 — 기능

### 다중 모달 입력

EurekAI는 프로필에 따라(기본적으로 어린이 및 청소년에 대해 활성화) 4가지 유형의 소스를 받습니다:

- **파일 가져오기** — JPG, PNG 또는 PDF 파일은 `mistral-ocr-latest`으로 처리(인쇄된 텍스트, 표, 손글씨)되며, TXT/MD 같은 텍스트 파일은 직접 가져옵니다.
- **자유 텍스트** — 텍스트를 입력하거나 붙여넣습니다. 저장 전 프로필에 따라 검열됩니다(활성화된 경우).
- **음성 입력** — 브라우저에서 오디오를 녹음합니다. `voxtral-mini-latest`로 전사됩니다. `language="fr"` 파라미터가 인식 최적화를 돕습니다.
- **웹 / URL** — 하나 이상의 URL을 붙여넣어 직접 스크래핑(Readability + Lightpanda, JS 페이지용)하거나 키워드를 입력해 Agent Mistral로 웹 검색을 수행합니다. 단일 입력 필드에서 URL과 키워드를 자동 분리하며, 각 결과는 독립적인 소스가 됩니다.

### AI 콘텐츠 생성

생성되는 학습 자료 유형은 7가지입니다:

| 생성기 | 모델 | 출력 |
|---|---|---|
| **복습 노트** | `mistral-large-latest` | 제목, 요약, 핵심 포인트, 어휘, 인용문, 에피소드 |
| **플래시카드** | `mistral-large-latest` | 출처 참조가 포함된 Q/A 카드(개수 설정 가능) |
| **객관식 퀴즈** | `mistral-large-latest` | 다지선다형 질문, 해설, 적응형 오답 복습(개수 설정 가능) |
| **빈칸 문제** | `mistral-large-latest` | 힌트가 포함된 문장 완성 문제, Levenshtein을 이용한 관용적 검증 |
| **팟캐스트** | `mistral-large-latest` + Voxtral TTS | 2인 음성 스크립트 → MP3 오디오 |
| **일러스트레이션** | Agent `mistral-large-latest` | `image_generation` 도구를 통한 교육용 이미지 |
| **음성 퀴즈** | `mistral-large-latest` + Voxtral TTS + STT | 질문 TTS → 응답 STT → AI 검증 |

### 채팅 기반 AI 튜터

수업 자료에 완전 접근 가능한 대화형 튜터:

- `mistral-large-latest` 사용
- **도구 호출**: 대화 중에 노트, 플래시카드, 퀴즈 또는 빈칸 문제를 생성할 수 있음
- 코스별 50 메시지의 히스토리 유지
- 프로필에 따라 콘텐츠 검열 적용

### 자동 라우터

라우터는 콘텐츠 분석을 위해 `mistral-small-latest`을 사용하여 가장 적합한 7가지 생성기 중 제안을 합니다. 인터페이스는 실시간 진행 상태를 표시합니다: 먼저 분석 단계, 그다음 개별 생성 단계(취소 가능).

### 적응형 학습

- **퀴즈 통계**: 문제별 시도 횟수 및 정답률 추적
- **퀴즈 복습**: 약한 개념을 겨냥한 5-10개의 새 질문 생성
- **지시 감지**: "Je sais ma leçon si je sais..." 같은 복습 지시를 감지하여 호환되는 텍스트 생성기(노트, 플래시카드, 퀴즈, 빈칸 문제)에 우선순위 부여

### 보안 및 부모 통제

- **4개 연령 그룹**: 어린이(≤10), 청소년(11-15), 학생(16-25), 성인(26+)
- **콘텐츠 검열**: `mistral-moderation-latest` 사용, 10개 카테고리 제공, 어린이/청소년 기본 차단 5개 카테고리(`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`). 설정에서 프로필별 커스터마이즈 가능.
- **부모 PIN**: SHA-256 해시 사용, 15세 미만 프로필에 필요. 프로덕션에서는 솔트가 있는 느린 해시(Argon2id, bcrypt) 권장.
- **채팅 제한**: 16세 미만 기본 비활성화, 부모가 활성화 가능

### 다중 프로필 시스템

- 이름, 나이, 아바타, 언어 선호를 갖는 다중 프로필
- 프로필에 연결된 프로젝트는 `profileId`으로 관리
- 계층 삭제: 프로필 삭제 시 해당 프로필의 모든 프로젝트가 삭제됨

### 다중 TTS 제공자 및 개인화 음성

- **Mistral Voxtral TTS**(기본): `voxtral-mini-tts-latest`, 추가 키 불필요
- **ElevenLabs**(대안): `eleven_v3`, 자연스러운 음성, `ELEVENLABS_API_KEY` 필요
- 제공자는 앱 설정에서 변경 가능
- **개인화 음성**: 부모는 Mistral Voices API(오디오 샘플로부터)를 통해 자체 음성을 생성하고 호스트/게스트 역할에 할당할 수 있으며, 팟캐스트와 음성 퀴즈는 부모 목소리로 재생되어 학습 경험을 더 몰입감 있게 만듭니다
- 구성 가능한 두 음성 역할: **호스트**(주 내레이터) 및 **게스트**(팟캐스트의 두 번째 음성)
- 설정에서 언어별로 필터 가능한 Mistral 음성 전체 카탈로그 제공

### 국제화

- 인터페이스 9개 언어 지원: fr, en, es, pt, it, nl, de, hi, ar
- AI 프롬프트 15개 언어 지원: fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv
- 프로필별 언어 설정 가능

## 컨테이너로 배포

이미지는 **GitHub Container Registry**에 게시되어 있습니다:

```bash
# Télécharger l'image
podman pull ghcr.io/jls42/eurekai:latest

# Lancer EurekAI
mkdir -p ./data
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=<your_api_key> \
  -e ELEVENLABS_API_KEY=<your_api_key> \
  -v ./data:/app/output:U \
  -p 3000:3000 \
  ghcr.io/jls42/eurekai:latest
# → http://localhost:3000
```

> **`:U`**는 볼륨 권한을 자동으로 조정하는 Podman rootless 플래그입니다.
> **`ELEVENLABS_API_KEY`**는 선택적입니다 (대체 TTS).

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

## 아키텍처 결정

| 결정 | 근거 |
|---|---|
| **React/Vue 대신 Alpine.js** | 최소한의 오버헤드, Vite로 컴파일된 TypeScript와 가벼운 반응성. 속도가 중요한 해커톤에 적합. |
| **JSON 파일로의 영속성** | 의존성 없음, 즉시 시작 가능. 설정할 데이터베이스 없음 — 곧바로 실행. |
| **Vite + Handlebars** | 두 세계의 장점: 개발용 빠른 HMR, 코드 정리를 위한 HTML partials, Tailwind JIT. |
| **중앙화된 프롬프트** | 모든 AI 프롬프트는 `prompts.ts`에 있음 — 언어/연령대별로 반복, 테스트 및 조정하기 쉬움. |
| **다중 생성 시스템** | 각 생성물은 고유 ID를 가진 독립 객체 — 수업당 여러 복습지, 퀴즈 등을 허용. |
| **연령별 맞춤 프롬프트** | 어휘, 복잡성, 어조가 다른 4개의 연령 그룹 — 동일한 콘텐츠라도 학습자에 따라 다르게 가르침. |
| **에이전트 기반 기능** | 이미지 생성 및 웹 검색은 임시 Mistral 에이전트를 사용 — 자동 정리로 깔끔한 수명 주기. |
| **지능형 URL 스크래핑** | 단일 입력란이 URL과 키워드를 혼합해서 허용 — URL은 Readability(정적 페이지)로 스크랩하고 폴백으로 Lightpanda(JS/SPA 페이지), 키워드는 Mistral 에이전트의 web_search를 트리거. 각 결과는 독립 소스를 생성. |
| **다중 TTS 제공자** | 기본값 Mistral Voxtral TTS(추가 키 불필요), 대체로 ElevenLabs — 재시작 없이 구성 가능. |
