## The story — Why EurekAI?

**EurekAI** was born during the [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online) ([official site](https://worldwide-hackathon.mistral.ai/)) (March 2026). I needed a topic — and the idea came from something very concrete: I regularly prepare tests with my daughter, and I thought it should be possible to make that more fun and interactive thanks to AI.

The goal: take **any input** — a photo of the textbook, copied-and-pasted text, a voice recording, a web search — and transform it into **study notes, flashcards, quizzes, podcasts, fill-in-the-blank texts, illustrations, and more**. All powered by Mistral AI's models, making it a solution naturally suited for French-speaking students.

The project was started during the hackathon, then continued and expanded afterwards. The entire codebase is generated by AI — mainly [Claude Code](https://docs.anthropic.com/en/docs/claude-code), with some contributions via [Codex](https://openai.com/index/introducing-codex/).

## Deep dive — Features

### Multi-modal input

EurekAI accepts 4 types of sources, moderated according to the profile (enabled by default for child and teen):

- **OCR Upload** — JPG, PNG or PDF files processed by `mistral-ocr-latest`. Handles printed text, tables and handwriting.
- **Free text** — Type or paste any content. Moderated before storage if moderation is active.
- **Voice input** — Record audio in the browser. Transcribed by `voxtral-mini-latest`. The `language="fr"` setting optimizes recognition.
- **Web search** — Enter a query. A temporary Mistral Agent with the `web_search` tool fetches and summarizes the results.

### AI content generation

Seven types of learning material generated:

| Generator | Model | Output |
|---|---|---|
| **Study note** | `mistral-large-latest` | Title, summary, key points, vocabulary, quotes, anecdote |
| **Flashcards** | `mistral-large-latest` | Q/A cards with references to sources (configurable number) |
| **Multiple-choice quiz** | `mistral-large-latest` | Multiple-choice questions, explanations, adaptive review (configurable number) |
| **Fill-in-the-blank texts** | `mistral-large-latest` | Sentences to complete with hints, tolerant validation (Levenshtein) |
| **Podcast** | `mistral-large-latest` + Voxtral TTS | 2-voice script → MP3 audio |
| **Illustration** | Agent `mistral-large-latest` | Educational image via the `image_generation` tool |
| **Voice quiz** | `mistral-large-latest` + Voxtral TTS + STT | TTS questions → STT answer → AI verification |

### AI tutor via chat

A conversational tutor with full access to course documents:

- Uses `mistral-large-latest`
- Tool calls: can generate study notes, flashcards, quizzes or fill-in-the-blank texts during the conversation
- 50-message history per course
- Content moderation if enabled for the profile

### Automatic router

The router uses `mistral-small-latest` to analyze source content and propose the most relevant generators among the 7 available. The UI shows real-time progress: first an analysis phase, then individual generations with possible cancellation.

### Adaptive learning

- Quiz statistics: tracking attempts and accuracy per question
- Quiz review: generates 5–10 new questions targeting weak concepts
- Instruction detection: detects revision instructions ("I know my lesson if I know...") and prioritizes them in compatible textual generators (study note, flashcards, quiz, fill-in-the-blank texts)

### Security & parental control

- 4 age groups: child (≤10), teen (11–15), student (16–25), adult (26+)
- Content moderation: `mistral-moderation-latest` with 5 categories blocked for child/teen (`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`), no restrictions for student/adult
- Parental PIN: SHA-256 hash, required for profiles under 15. For production deployment, use a slow salted hash (Argon2id, bcrypt).
- Chat restrictions: AI chat disabled by default for under 16s, can be enabled by parents

### Multi-profile system

- Multiple profiles with name, age, avatar, language preferences
- Projects linked to profiles via `profileId`
- Cascading deletion: deleting a profile deletes all its projects

### Multi-provider TTS

- **Mistral Voxtral TTS** (default): `voxtral-mini-tts-latest`, no extra key required
- **ElevenLabs** (alternative): `eleven_v3`, natural voices, requires `ELEVENLABS_API_KEY`
- Provider configurable in the app settings

### Internationalization

- Interface available in 9 languages: fr, en, es, pt, it, nl, de, hi, ar
- AI prompts support 15 languages (fr, en, es, de, it, pt, nl, ja, zh, ko, ar, hi, pl, ro, sv)
- Language configurable per profile

## Model reference

| Model | Usage | Why |
|---|---|---|
| `mistral-large-latest` | Study note, Flashcards, Podcast, Quiz, Fill-in-the-blank texts, Chat, Voice quiz verification, Image Agent, Web Search Agent, Instruction detection | Best multilingual + instruction following |
| `mistral-ocr-latest` | Document OCR | Printed text, tables, handwriting |
| `voxtral-mini-latest` | Speech recognition (STT) | Multilingual STT, optimized with `language="fr"` |

## Quick start

```bash
# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install

# Configurer les clés API
cp .env.example .env
# Éditez .env avec vos clés :
#   MISTRAL_API_KEY=votre_clé_ici           (requis)
#   ELEVENLABS_API_KEY=votre_clé_ici        (optionnel, TTS alternatif)
#   SONAR_TOKEN=...                          (optionnel, CI SonarCloud uniquement)

# Lancer le développement
npm run dev
# → Backend :  http://localhost:3000 (API)
# → Frontend : http://localhost:5173 (serveur Vite avec HMR)
```

> **Note**: Mistral Voxtral TTS is the default provider — no additional key required beyond `MISTRAL_API_KEY`. ElevenLabs is an alternative TTS provider configurable in the settings.

## Architectural decisions

| Decision | Rationale |
|---|---|
| **Alpine.js rather than React/Vue** | Minimal footprint, lightweight reactivity with TypeScript compiled by Vite. Perfect for a hackathon where speed matters. |
| **Persistence in JSON files** | Zero dependencies, instant startup. No database to configure — you start and go. |
| **Vite + Handlebars** | The best of both worlds: fast HMR for development, HTML partials for code organization, Tailwind JIT. |
| **Centralized prompts** | All AI prompts in `prompts.ts` — easy to iterate, test and adapt by language/age group. |
| **Multi-generation system** | Each generation is an independent object with its own ID — allows multiple worksheets, quizzes, etc. per course. |
| **Age-adapted prompts** | 4 age groups with different vocabulary, complexity and tone — the same content teaches differently depending on the learner. |
| **Agent-based features** | Image generation and web search use temporary Mistral Agents — a clean lifecycle with automatic cleanup. |
| **Multi-provider TTS** | Mistral Voxtral TTS by default (no additional key), ElevenLabs as an alternative — configurable without restart. |
