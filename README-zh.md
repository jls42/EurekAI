## 故事 — 为什么 EurekAI？

**EurekAI** 起源于 [Mistral AI Worldwide Hackathon](https://luma.com/mistralhack-online)（[官方网站](https://worldwide-hackathon.mistral.ai/)）(2026 年 3 月)。我需要一个题目 —— 想法来自一个非常具体的场景：我经常和我女儿一起准备考试，我觉得应该可以利用人工智能让这个过程更有趣、更具互动性。

目标：接受任意输入 —— 课堂照片、复制粘贴的文本、语音录音、网页搜索 —— 并将其转换为 **复习笔记、抽认卡、测验、播客、填空练习、插图等**。所有内容由法国的 Mistral AI 模型驱动，因此天然适合讲法语的学生。

最初的[原型](https://github.com/jls42/worldwide-hackathon.mistral.ai) 在黑客松的 48 小时内作为围绕 Mistral 服务的概念验证被开发出来 —— 已可运行但功能有限。从那时起，EurekAI 成为一个真正的项目：填空题、练习导航、网页抓取、可配置的家长审查、深入的代码复审等等。全部代码由人工智能生成 —— 主要是 [Claude Code](https://code.claude.com/)，并有少量通过 [Codex](https://openai.com/codex/) 和 [Gemini CLI](https://geminicli.com/) 的贡献。

## 深入功能

### 多模态输入

EurekAI 接受 4 种来源类型，根据配置的档案进行审查（默认对儿童和青少年启用）：

- **导入文件** — JPG、PNG 或通过 `mistral-ocr-latest` 处理的 PDF（打印文本、表格、手写），或直接导入的文本文件（TXT、MD）。
- **自由文本** — 输入或粘贴任意内容。若启用审查，存储前会进行审查。
- **语音输入** — 在浏览器中录音。由 `voxtral-mini-latest` 转写。参数 `language="fr"` 优化识别效果。
- **网页 / URL** — 粘贴一个或多个 URL 以直接抓取内容（Readability + Lightpanda 处理 JS 页面），或输入关键词通过 Agent Mistral 进行网页搜索。输入框同时接受 URL 和关键词，会自动分离，每个结果会成为独立来源。

### AI 内容生成

生成七类学习材料：

| 生成器 | 模型 | 输出 |
|---|---|---|
| **复习笔记** | `mistral-large-latest` | 标题、摘要、要点、词汇、引用、趣闻 |
| **抽认卡** | `mistral-large-latest` | 带来源引用的问答卡（可配置数量） |
| **选择题测验** | `mistral-large-latest` | 多项选择题、解析、自适应复习（可配置数量） |
| **填空题** | `mistral-large-latest` | 带提示的待填句子，容错验证（Levenshtein） |
| **播客** | `mistral-large-latest` + Voxtral TTS | 双声部脚本 → MP3 音频 |
| **插图** | Agent `mistral-large-latest` | 通过工具 `image_generation` 生成的教育图片 |
| **口语测验** | `mistral-large-latest` + Voxtral TTS + STT | TTS 朗读题目 → STT 识别回答 → AI 验证 |

### 聊天型 AI 导师

一个可以访问全部课程文档的会话导师：

- 使用 `mistral-large-latest`
- 工具调用：在对话中可生成复习笔记、抽认卡、测验或填空题
- 每门课程保留 50 条消息的历史
- 若档案启用，则对话内容会进行审查

### 自动路由器

路由器使用 `mistral-small-latest` 分析来源内容并在 7 种生成器中推荐最相关的生成器。界面实时显示进度：首先为分析阶段，然后展示各项生成任务并允许取消。

### 自适应学习

- **测验统计**：跟踪每题的尝试次数和准确率
- **测验复习**：生成 5-10 道针对薄弱概念的新题
- **指令检测**：识别复习指令（“如果我会……就说明我会这课”）并在兼容的文本生成器（笔记、抽认卡、测验、填空）中优先使用

### 安全与家长控制

- **4 个年龄组**：儿童（≤10 岁）、青少年（11-15）、学生（16-25）、成人（26+）
- **内容审查**：使用 `mistral-moderation-latest`，提供 10 个可选类别，儿童/青少年默认屏蔽 5 类（`sexual`, `hate_and_discrimination`, `violence_and_threats`, `selfharm`, `jailbreaking`）。各档案在设置中可自定义被屏蔽类别。
- **家长 PIN**：SHA-256 哈希，针对 15 岁以下档案必需。若用于生产部署，应使用带盐的慢哈希（Argon2id、bcrypt）。
- **聊天限制**：16 岁以下默认禁用 AI 聊天，可由父母启用

### 多档案系统

- 多个档案，包含姓名、年龄、头像、语言偏好
- 项目与档案通过 `profileId` 关联
- 级联删除：删除档案将删除其所有项目

### 多提供商 TTS 与自定义语音

- **Mistral Voxtral TTS**（默认）：`voxtral-mini-tts-latest`，无需额外密钥
- **ElevenLabs**（可选）：`eleven_v3`，自然语音，需要 `ELEVENLABS_API_KEY`
- 可在应用设置中配置提供商
- **自定义语音**：父母可通过 Mistral Voices API（使用音频样本）创建自己的语音，并分配给主/客角色 —— 播客和口语测验即可使用父母的声音，使儿童体验更具沉浸感
- 两个可配置语音角色：**主讲**（主要叙述）和 **嘉宾**（播客的第二声部）
- 在设置中可查看完整的 Mistral 语音目录，并按语言过滤

### 国际化

- 界面支持 9 种语言：fr、en、es、pt、it、nl、de、hi、ar
- AI prompts 支持 15 种语言（fr、en、es、de、it、pt、nl、ja、zh、ko、ar、hi、pl、ro、sv）
- 语言可按档案配置

## 技术栈

| 层 | 技术 | 角色 |
|---|---|---|
| **运行时** | Node.js + TypeScript 6.x | 服务器与类型安全 |

| **测试** | Vitest | 单元测试 —— 覆盖率由 SonarCloud 测量 |
| **持久化** | JSON 文件 | 无额外依赖的存储 |

## 模型参考

| 模型 | 用途 | 为什么 |
|---|---|---|
| `mistral-large-latest` | 笔记、抽认卡、播客、测验、填空、聊天、口语测验校验、图像 Agent、网页检索 Agent、指令检测 | 最佳的多语言能力 + 指令跟随能力 |
| `mistral-ocr-latest` | 文档 OCR | 打印文本、表格、手写识别 |
| `voxtral-mini-latest` | 语音识别（STT） | 多语言 STT，配合 `language="fr"` 优化 |
| `voxtral-mini-tts-latest` | 语音合成（TTS） | 播客、口语测验、朗读功能 |
| `mistral-moderation-latest` | 内容审查 | 对儿童/青少年默认屏蔽 5 类（+ 防绕过） |
| `mistral-small-latest` | 自动路由器 | 快速分析内容用于路由决策 |
| `eleven_v3` (ElevenLabs) | 语音合成（TTS 可选） | 自然语音，可配置的替代方案 |

## 快速开始

```bash
# Cloner le dépôt
git clone https://github.com/jls42/EurekAI.git
cd EurekAI

# Installer les dépendances
npm install

# Configurer les clés API
cp .env.example .env
# Éditez .env avec vos clés :
#   MISTRAL_API_KEY=<your_api_key>           (requis)
#   ELEVENLABS_API_KEY=<your_api_key>        (optionnel, TTS alternatif)
#   SONAR_TOKEN=...                          (optionnel, CI SonarCloud uniquement)

# Lancer le développement
npm run dev
# → Backend :  http://localhost:3000 (API)
# → Frontend : http://localhost:5173 (serveur Vite avec HMR)
```

> **注意**：Mistral Voxtral TTS 为默认提供商 —— 除了 `MISTRAL_API_KEY` 不需要额外密钥。ElevenLabs 是可在设置中配置的可选 TTS 提供商。

## 使用容器部署

镜像发布在 **GitHub Container Registry**：

```bash
# Télécharger l'image
podman pull ghcr.io/jls42/eurekai:latest

# Lancer EurekAI
mkdir -p ./data
podman run -d --name eurekai \
  -e MISTRAL_API_KEY=<your_api_key> \
  -e ELEVENLABS_API_KEY=<your_api_key> \
  -v ./data:/app/output:U \
  -p 3000:3000 \
  ghcr.io/jls42/eurekai:latest
# → http://localhost:3000
```

> **`:U`** 是 Podman 无 root 标志，会自动调整卷权限。
> **`ELEVENLABS_API_KEY`** 为可选项（可选 TTS）。

```bash
# Build local
podman build -t eurekai -f Containerfile .

# Publier sur ghcr.io (mainteneurs)
./scripts/publish-ghcr.sh
```

## 架构决策

| 决策 | 理由 |
|---|---|
| **使用 Alpine.js 而非 React/Vue** | 占用最小，配合 Vite 编译的 TypeScript 提供轻量响应性。非常适合重视速度的 hackathon。 |
| **使用 JSON 文件持久化** | 零依赖，立即启动。无需配置数据库 — 即开即用。 |
| **Vite + Handlebars** | 两全其美：开发时快速 HMR，HTML partials 用于代码组织，Tailwind JIT。 |
| **集中式提示（Prompts）** | 所有 AI 提示位于 `prompts.ts` — 易于迭代、测试并按语言/年龄组调整。 |
| **多生成系统** | 每个生成都是具有自己 ID 的独立对象 — 允许每门课程拥有多份复习资料、测验等。 |
| **按年龄适配的提示** | 4 个年龄组，词汇、复杂度和语气不同 — 相同内容可根据学习者不同方式教学。 |
| **基于 Agents 的功能** | 图像生成与网络搜索使用临时 Mistral Agents — 生命周期清晰并自动清理。 |
| **智能 URL 抓取** | 单一字段同时接受 URL 与关键词 — URL 使用 Readability 抓取（静态页面），回退到 Lightpanda（JS/SPA 页面），关键词则触发 Mistral web_search Agent。每个结果创建一个独立来源。 |
| **多 TTS 提供商** | 默认使用 Mistral Voxtral TTS（无需额外密钥），可选 ElevenLabs — 无需重启即可配置。 |
